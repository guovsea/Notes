## 描述

右键添加子节点并聚焦到正在编辑的子节点上
## 内容

在 `QListWidget` 中支持右键添加子节点并聚焦到正在编辑的子节点上，可以通过以下几个步骤实现：

1. **启用自定义上下文菜单**：首先，需要为 `QListWidget` 设置自定义上下文菜单策略，这样当用户右键点击时，可以显示自定义的菜单。

    ```cpp
    ui->listWidget->setContextMenuPolicy(Qt::CustomContextMenu);
    ```

2. **重写 `contextMenuEvent`**：重写 `QListWidget` 的 `contextMenuEvent` 函数来创建和显示上下文菜单，并在菜单中添加添加子节点的选项。

    ```cpp
    void ListWidget::contextMenuEvent(QContextMenuEvent *event) {
        QMenu *menu = new QMenu(this);
        QAction *addAction = menu->addAction(tr("Add Subitem"));
        QAction *editAction = menu->addAction(tr("Edit Item"));
        QAction *selectedAction = menu->exec(event->globalPos());
        if (selectedAction == addAction) {
            // 添加子节点的逻辑
        } else if (selectedAction == editAction) {
            // 编辑当前节点的逻辑
        }
        delete menu;
    }
    ```

3. **添加子节点**：在添加子节点的槽函数中，获取当前选中的项，并在其下添加新的子节点。

    ```cpp
    void ListWidget::addSubitem() {
        QListWidgetItem *currentItem = ui->listWidget->currentItem();
        if (currentItem) {
            QListWidgetItem *newItem = new QListWidgetItem(tr("New Subitem"), ui->listWidget);
            newItem->setParentItem(currentItem);
            // 设置新的子节点为当前项，以便编辑
            ui->listWidget->setCurrentItem(newItem);
        }
    }
    ```

4. **聚焦到编辑的子节点**：在添加子节点后，使用 `setCurrentItem` 函数将新添加的子节点设置为当前项，这样它就会被聚焦。

    ```cpp
    ui->listWidget->setCurrentItem(newItem);
    ```
    
5. **连接信号和槽**：确保将添加子节点的槽函数连接到上下文菜单中相应的动作上。

 ```cpp
    connect(addAction, &QAction::triggered, this, &ListWidget::addSubitem);
    ```


这样，当用户在 `QListWidget` 上右键点击时，就可以通过自定义的上下文菜单添加子节点，并且新添加的子节点会自动获得焦点，允许用户立即开始编辑。
