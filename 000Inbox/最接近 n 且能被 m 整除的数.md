## 问题描述

给定两个整数 `n` 和 `m`（`m ≠ 0`），找到最接近 `n` 且能被 `m` 整除的数。若有多个满足条件的数（即到 `n` 的距离相同），则返回绝对值较大的那个。

---

## 核心思路：基于整数除法的几何意义

### 关键观察：C++ 整数除法向零截断

在 C++ 中，整数除法 **向零截断（truncation toward zero）**，即：

- 正数除法向下取整：`13 / 4 = 3`
- 负数除法向上取整：`-15 / 6 = -2`（而非数学上的 `-3`）

这意味着，对于任意 `n` 和 `m`，商 `q = n / m` 所对应的乘积 `m * q` **一定比 `n` 更靠近数轴原点**（或恰好等于 `n`）。换句话说，`m * q` 永远落在原点到 `n` 的这一侧。

> **例**：`n = 13, m = 4` → `q = 3` → `m * q = 12`，12 比 13 更靠近原点。  
> **例**：`n = -15, m = 6` → `q = -2` → `m * q = -12`，-12 比 -15 更靠近原点。

### 第一个候选：`n1 = m * q`

由上述性质，`n1` 就是原点到 `n` 这条路径上、离 `n` 最近的 `m` 的整倍数。

### 第二个候选：向远离原点的方向多走一步

问题转化为：**找到在 `n` 远离原点方向上的另一个 `m` 的整倍数**。

这可以通过对 `|q|` 加 1 来实现。理解除法的几何意义：

> `n / m = q` 的含义是：从原点出发，以步长 `|m|` 行走 `|q|` 步（方向由 `q` 的符号决定），即可到达 `n` 附近。剩余的余数就是 `n` 距 `m * q` 的偏移量。

要走到 `n` **远离原点**那一侧的整倍数，只需在 `|q|` 步的基础上 **多走一步**：

- **`n` 与 `m` 同号** → `q > 0` → `|q| + 1 = q + 1` → 第二个候选为 `m * (q + 1)`
- **`n` 与 `m` 异号** → `q < 0` → `|q| + 1 = -(q - 1) = q - 1` 的绝对值 → 第二个候选为 `m * (q - 1)`

> 这就是代码中 `(n * m) > 0 ? m * (q + 1) : m * (q - 1)` 的由来——用 `n * m` 的符号判断同号/异号，从而决定对 `q` 加 1 还是减 1（本质上都是对 `|q|` 加 1）。

### 取更优候选

比较 `n1` 和 `n2` 与 `n` 的距离：

- 若 `|n - n1| < |n - n2|`，返回 `n1`
- 若 `|n - n1| > |n - n2|`，返回 `n2`
- 若距离相等，返回绝对值更大的那个（即 `n2`，因为它在远离原点方向）

由于代码中 `n2` 本身就是远离原点方向的候选，其绝对值必然 ≥ `n1` 的绝对值，因此距离相等时直接返回 `n2` 即可。

## 图解示例

### 示例 1：`n = 13, m = 4`

```
数轴: ...  8 ---- 12 ---- 13 ---- 16 ...
                    n1      n       n2

q = 13 / 4 = 3 (向零截断)
n1 = 4 × 3  = 12    (靠近原点侧)
n2 = 4 × 4  = 16    (远离原点侧，|q|+1 = 3+1 = 4)

|13 - 12| = 1 < |13 - 16| = 3  →  返回 12 ✅
```

### 示例 2：`n = -15, m = 6`

```
数轴: ... -18 ---- -15 ---- -12 ---- 0 ...
         n2        n        n1

q = -15 / 6 = -2  (向零截断，而非数学上的 -3)
n1 = 6 × (-2) = -12   (靠近原点侧)
n2 = 6 × (-3) = -18   (远离原点侧，|q|+1 = 2+1 = 3, q-1 = -3)

|-15 - (-12)| = 3 == |-15 - (-18)| = 3  →  距离相等
返回绝对值更大者 → -18 ✅
```

## 完整代码

```cpp
#include <iostream>
#include <cstdlib>
using namespace std;

int closestNumber(int n, int m) {
    int q = n / m;

    // 候选1：靠近原点侧的 m 的整倍数（由截断除法保证）
    int n1 = m * q;

    // 候选2：远离原点侧的 m 的整倍数（对 |q| 加 1）
    int n2 = (n * m > 0) ? m * (q + 1)   // 同号 → q > 0 → |q|+1 即 q+1
                         : m * (q - 1);   // 异号 → q < 0 → |q|+1 即 -(q-1)

    // 取距离 n 更近者；距离相等时 n2 绝对值更大，天然满足要求
    if (abs(n - n1) < abs(n - n2))
        return n1;
    return n2;
}

int main() {
    cout << closestNumber(13, 4)   << endl;   // 12
    cout << closestNumber(-15, 6)  << endl;   // -18
    cout << closestNumber(10, 3)   << endl;   // 9
    cout << closestNumber(-10, 3)  << endl;   // -9
    return 0;
}
```

## 要点总结

| 要点 | 说明 |
|:---|:---|
| **截断除法的性质** | `q = n / m` 向零截断，保证 `m * q` 在原点到 `n` 之间 |
| **几何视角** | `q` 代表从原点以步长 `\|m\|` 走了 `\|q\|` 步，还差一点到 `n` |
| **第二步候选** | 对 `\|q\|` 加 1 步，跳到 `n` 远离原点那一侧的整倍数 |
| **同号/异号的判断** | 决定对 `q` 执行 `+1` 还是 `-1`（本质都是 `\|q\|+1`） |
| **平局处理** | 距离相等时返回 `n2`，因其绝对值更大，天然满足题目要求 |
