# 05 - 能带理论与载流子统计

> 本章深入讲解半导体的能带结构、费米能级概念，以及掺杂半导体的载流子统计理论。

---

## 能带结构

### 从原子轨道到能带

#### 孤立原子的能级

在孤立原子中，电子只能处于一系列**分立的能级**上：
- 1s, 2s, 2p, 3s, 3p, ...
- 每个能级可容纳有限数目的电子（泡利不相容原理）

#### 晶体中的能带形成

当N个原子靠近形成晶体时：
- 每个原子能级分裂成N个紧密排列的能级
- 由于N极大（~10²³ cm⁻³），这些能级形成**准连续的能带**

**能级分裂的物理原因**：
- 相邻原子的波函数交叠
- 量子力学隧穿效应
- 泡利不相容原理要求电子不能占据相同状态

```
孤立原子能级          双原子能级           晶体中能带
    |                    |                  ████████
 2p |  ●                ●   ●              ████████
    |                  /     \              ████████
 2s |  ●              ●───────●            ████████
    |                                      ────────
 1s |  ●                ●   ●              ████████
    |                    \_/               ████████

N个能级→N个能级→准连续能带
```

### 导带、价带、禁带

#### 导带（Conduction Band）

**定义**：晶体中电子可以自由移动参与导电的能带。

**特征**：
- 导带底（$E_c$）：导带的最低能量
- 导带中电子浓度很低（非简并情况下）
- 电子可在外电场作用下获得动量

#### 价带（Valence Band）

**定义**：0K时被电子完全填满的最高能带。

**特征**：
- 价带顶（$E_v$）：价带的最高能量
- 0K时完全填满，无导电能力
- 存在空穴时可导电

#### 禁带（Band Gap）

**定义**：导带底与价带顶之间的能量区域，不存在允许的电子态。

**禁带宽度**：
$$E_g = E_c - E_v$$

**典型值**（300K）：
| 材料 | $E_g$ (eV) | 带隙类型 |
|------|-----------|---------|
| Ge | 0.66 | 间接带隙 |
| Si | 1.12 | 间接带隙 |
| GaAs | 1.42 | 直接带隙 |
| SiC (4H) | 3.26 | 间接带隙 |
| GaN | 3.4 | 直接带隙 |

**直接带隙 vs 间接带隙**：
- **直接带隙**：导带底和价带顶在k空间同一点（如GaAs）
  - 电子跃迁不需要声子参与
  - 发光效率高
- **间接带隙**：导带底和价带顶在k空间不同点（如Si）
  - 电子跃迁需要声子参与
  - 发光效率低

### 硅的能带结构详解

#### E-k关系

硅的导带底位于[100]方向的Δ点附近，不在k=0处：
- 6个等价的导带谷
- 等能面为旋转椭球

**纵向有效质量**：$m_l^* \approx 0.98 m_0$
**横向有效质量**：$m_t^* \approx 0.19 m_0$
**态密度有效质量**：$m_{dos}^* = (m_l^* m_t^{*2})^{1/3} \approx 0.33 m_0$

**电导有效质量**：
$$\frac{1}{m_c^*} = \frac{1}{3}\left(\frac{1}{m_l^*} + \frac{2}{m_t^*}\right) \approx \frac{1}{0.26 m_0}$$

#### 重空穴与轻空穴

价带顶存在两个子带：
- **重空穴带**：有效质量大 ($m_{hh}^* \approx 0.49 m_0$)，曲率小
- **轻空穴带**：有效质量小 ($m_{lh}^* \approx 0.16 m_0$)，曲率大
- **自旋-轨道分裂带**：能量较低，影响较小

**态密度有效质量**：
$$m_p^* = (m_{hh}^{*3/2} + m_{lh}^{*3/2})^{2/3} \approx 0.55 m_0$$

## 费米能级

### 费米-狄拉克分布

电子服从**费米-狄拉克统计**：

$$f(E) = \frac{1}{1 + \exp\left(\frac{E - E_F}{k_BT}\right)}$$

其中：
- $f(E)$：能量为E的状态被电子占据的概率
- $E_F$：费米能级（Fermi Level）
- $k_B$：玻尔兹曼常数
- $T$：绝对温度

#### 费米分布的特征

**在$E = E_F$时**：
$$f(E_F) = \frac{1}{2}$$

**在$T = 0$K时**：
$$f(E) = \begin{cases} 1, & E < E_F \\ 0, & E > E_F \end{cases}$$

电子填充到$E_F$为止，高于$E_F$的状态全部空着。

**在$T > 0$K时**：
- 分布函数在$E_F$附近平滑过渡
- 过渡区域的宽度约为$k_BT$（~26 meV @ 300K）

```

    │    ████

    │            ████

            E_F

  1 │████████
    │        ████████

    │                        ████████
  0 │                                ████
    └────────────────────────────────→ E
                E_F

```

### 本征费米能级

#### 本征半导体中的$E_F$位置

在本征半导体中，$E_F$位于禁带中央附近：

$$E_i = \frac{E_c + E_v}{2} + \frac{k_BT}{2}\ln\left(\frac{N_v}{N_c}\right)$$

对于硅（$m_p^* \approx m_n^*$）：
$$E_i \approx \frac{E_c + E_v}{2} = E_{midgap}$$

本征费米能级几乎位于禁带正中央。

#### 本征载流子浓度

$$n_i = N_c \exp\left(-\frac{E_c - E_i}{k_BT}\right) = N_v \exp\left(-\frac{E_i - E_v}{k_BT}\right)$$

$$n_i = \sqrt{N_c N_v} \exp\left(-\frac{E_g}{2k_BT}\right)$$

## n型半导体费米能级

### 费米能级位置

在n型半导体中，施主杂质提供电子，使$E_F$向导带底移动：

$$E_F = E_c - k_BT \ln\left(\frac{N_c}{n}\right)$$

对于完全电离的n型半导体（$n \approx N_D$）：

$$E_F = E_c - k_BT \ln\left(\frac{N_c}{N_D}\right)$$

**数值示例**（Si, 300K）：
- $N_c = 2.8 \times 10^{19}$ cm⁻³
- 若$N_D = 10^{16}$ cm⁻³：
  $$E_c - E_F = 0.026 \times \ln(2800) \approx 0.21 \text{ eV}$$

### 费米能级与掺杂浓度的关系

| $N_D$ (cm⁻³) | $E_c - E_F$ (eV) | 位置描述 |
|-------------|-----------------|---------|
| $10^{15}$ | 0.27 | 接近本征 |
| $10^{16}$ | 0.21 | 导带底下方 |
| $10^{17}$ | 0.15 | 更靠近导带 |
| $10^{18}$ | 0.09 | 接近导带底 |
| $10^{19}$ | 0.03 | 进入导带（简并） |

### 费米能级与温度的关系

```
E_F
↑
E_c ┼─────────────────────
    │    \         ↗
    │     \      /
E_i ┼──────●────●──────── 本征费米能级
    │      /\  /
E_v ┼─────/  \/
    │    /    ↘
    └────────────────→ T
          T₁

- 低温：$E_F$在$E_D$和$E_c$之间
- 中温（饱和区）：$E_F$下降
- 高温（本征区）：$E_F \rightarrow E_i$
```

## p型半导体费米能级

### 费米能级位置

在p型半导体中，受主杂质接受电子，使$E_F$向价带顶移动：

$$E_F = E_v + k_BT \ln\left(\frac{N_v}{p}\right)$$

对于完全电离的p型半导体（$p \approx N_A$）：

$$E_F = E_v + k_BT \ln\left(\frac{N_v}{N_A}\right)$$

**数值示例**（Si, 300K）：
- $N_v = 1.0 \times 10^{19}$ cm⁻³
- 若$N_A = 10^{16}$ cm⁻³：
  $$E_F - E_v = 0.026 \times \ln(1000) \approx 0.18 \text{ eV}$$

### p型与n型的对比

```

       │                      │
E_v ───┴───            E_v ──┴───

E_F靠近导带              E_F靠近价带
```

## 电中性条件

### 一般形式的电中性条件

半导体中总的正电荷等于总的负电荷：

$$p + N_D^+ = n + N_A^-$$

其中：
- $p$：空穴浓度（正电荷）
- $N_D^+$：电离施主浓度（正电荷）
- $n$：电子浓度（负电荷）
- $N_A^-$：电离受主浓度（负电荷）

### 电离杂质浓度

**施主电离**：
$$N_D^+ = \frac{N_D}{1 + g_D \exp\left(\frac{E_F - E_D}{k_BT}\right)}$$

其中$g_D = 2$（自旋简并）。

**受主电离**：
$$N_A^- = \frac{N_A}{1 + g_A \exp\left(\frac{E_A - E_F}{k_BT}\right)}$$

其中$g_A = 4$（价带简并）。

### 求解步骤

1. 写出电中性条件
2. 代入载流子浓度公式：
   $$n = N_c \exp\left(-\frac{E_c - E_F}{k_BT}\right)$$
   $$p = N_v \exp\left(-\frac{E_F - E_v}{k_BT}\right)$$
3. 代入电离杂质公式
4. 求解关于$E_F$的方程
5. 代回求出n和p

## 质量作用定律

### 推导

由载流子浓度公式：
$$n \cdot p = N_c N_v \exp\left(-\frac{E_c - E_v}{k_BT}\right) = N_c N_v \exp\left(-\frac{E_g}{k_BT}\right) = n_i^2$$

**质量作用定律**：
$$n \cdot p = n_i^2$$

这是热平衡条件下的普遍关系，与掺杂无关。

### 应用

**对于n型半导体**：
$$n \approx N_D, \quad p = \frac{n_i^2}{N_D}$$

**对于p型半导体**：
$$p \approx N_A, \quad n = \frac{n_i^2}{N_A}$$

---

## 载流子浓度计算

### 载流子浓度与掺杂浓度的关系

**n型半导体**：
$$n_n = \frac{N_D}{2} + \sqrt{\left(\frac{N_D}{2}\right)^2 + n_i^2}$$

当$N_D \gg n_i$时：
$$n_n \approx N_D, \quad p_n \approx \frac{n_i^2}{N_D}$$

**p型半导体**：
$$p_p = \frac{N_A}{2} + \sqrt{\left(\frac{N_A}{2}\right)^2 + n_i^2}$$

当$N_A \gg n_i$时：
$$p_p \approx N_A, \quad n_p \approx \frac{n_i^2}{N_A}$$

### 补偿半导体

当同时存在施主和受主时：

**补偿类型判断**：
- 若$N_D > N_A$：n型，$n \approx N_D - N_A$
- 若$N_A > N_D$：p型，$p \approx N_A - N_D$

**电中性条件**：
$$n + N_A^- = p + N_D^+$$

**有效掺杂浓度**：
$$N_{eff} = |N_D - N_A|$$

## 简并半导体

### 简并条件

当掺杂浓度足够高时，费米能级进入导带（n型）或价带（p型）：

**n型简并条件**：
$$E_F > E_c \quad \text{或} \quad N_D > N_c$$

**p型简并条件**：
$$E_F < E_v \quad \text{或} \quad N_A > N_v$$

硅中的简并浓度阈值约为$10^{19}$ cm⁻³。

### 带隙变窄效应

**物理原因**：
1. **杂质能带形成**：高浓度下杂质波函数交叠，形成杂质能带
2. **多体效应**：载流子-载流子相互作用
3. **屏蔽效应**：高载流子浓度对库仑势的屏蔽

**经验公式**：
$$\Delta E_g = A \left(\frac{N}{10^{18}}\right)^{1/3} \text{ (meV)}$$

对于硅，A ≈ 22 meV。

**影响**：
- 本征载流子浓度增加
- 发射结注入效率降低
- 需要修正载流子浓度公式

### 简并半导体的载流子统计

必须使用**费米-狄拉克积分**：
$$n = N_c F_{1/2}\left(\frac{E_F - E_c}{k_BT}\right)$$

其中费米-狄拉克积分定义为：
$$F_j(\eta) = \frac{1}{\Gamma(j+1)} \int_0^\infty \frac{x^j}{1 + \exp(x - \eta)} dx$$
