## 解答

```cpp

class Solution {
  public:
    string longestPalindrome(string s) {
        int L = s.size();
        // 记录遍历过程中dp[i][j]对应的字符串s[i][j]的长度
        // 初始值表示s[0]一定为回文字串
        int begin = 0;
        int len = 1;
        vector<vector<bool>> dp(L, vector<bool>(L, false));
        // 长度为1、2的字符串是否为回文
        dp[L - 1][L - 1] = true;
        for (int i = 0; i < L - 1; i++) {
            // 对角线和对角线右边的元素一起填充
            dp[i][i] = true;
            dp[i][i + 1] = s[i] == s[i + 1] ? true : false;
            if (dp[i][i + 1]) {
                begin = i;
                len = 2;
            }
        }
        // 从下往上填充，因为dp[i][j] = dp[i+1][j-1] && (s[i] == s[j]),
        // dp[i][j] 依赖于 dp[i+1][j-1],左下角
        for (int i = L - 2; i >= 0; i--) {
            for (int j = i + 2; j < L; j++) {
                dp[i][j] = dp[i + 1][j - 1] && (s[i] == s[j]);
                if (dp[i][j] && j - i + 1 > len) {
                    begin = i;
                    len = j - i + 1;
                }
            }
        }
        return s.substr(begin, len);
    }
};
```

**关键**

`dp[i][j]`表示的是 `s[i]`到 `s[j]`是**是否为回文字串**

**边界条件填充**：对角线`dp[i][i]`和`dp[i][i+1]，分别考虑长度为1的字串和长度为2的字串是否为回文字串

**状态转移**：`dp[i][j] = dp[i+1][j-1] && (s[i] == s[j])`

**特殊的填充方式**：因为 `dp[i][j]`依赖于`dp[i+1][j-1]`（右下角），因此最好的方式是从下往上、从左往右填充

## 题目

[力扣（LeetCode）官网 - 全球极客挚爱的技术成长平台](https://leetcode.cn/problems/longest-palindromic-substring/description/)

给你一个字符串 `s`，找到 `s` 中最长的回文子串。

如果字符串的反序与原始字符串相同，则该字符串称为回文字符串。
