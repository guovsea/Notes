双指针是一种常用于解决数组和字符串问题的算法技巧，它通常涉及到两个指针在数组或字符串上的移动，以达到特定的目标。以下是双指针的常见几种书写方式：

**快慢指针**

这种方式通常用于解决链表问题，其中一个指针（快指针）比另一个指针（慢指针）快一步或多步。快指针用于搜索，慢指针用于跟踪位置，从而实现某种特定目标，比如检测环、找到中间节点等。

**左右指针**

这种方式通常用于解决数组或字符串问题，其中左指针和右指针分别指向数组或字符串的两端，然后根据问题的要求移动这两个指针，通常在数组有序的情况下使用，例如查找两数之和、反转字符串等。

**滑动窗口**

滑动窗口是一种双指针的技巧，通常用于解决字符串和数组问题，其中一个指针维护窗口的起始位置，另一个指针用于扩展或缩小窗口。这种方法通常用于解决子数组或子字符串的问题，如找到最短子数组、最长无重复字符子串等。

对撞指针

对撞指针是一种在数组或字符串两侧操作的双指针技巧，通常用于有序数组或字符串中查找目标元素或满足特定条件的元素对。指针从两侧向中间靠拢，根据问题的要求移动它们。

## 快慢指针应用于链表

例如
- 寻找距离尾部第 K 个节点
- 判断有无环
- 寻找环入口
- 寻找公共尾部入口

1. 寻找环入口
   
   判断一个链表中是否有环，有则返回入环的第一个节点，无则返回`nullptr`

**思路**

第一次遍历：`fast = fast->next->next;` 如果fast遇到`nullptr`了，则说明没有环，返回nullptr。直到fast和slow相遇。

第二次遍历：fast = head;然后fast和slow每次分别直走一个节点，直到`fast`和`slow`相遇。相遇时所指向的节点极为所求节点。

```cpp
class Solution {
public:
    ListNode *detectCycle(ListNode *head) {
       ListNode* fast = head;
       ListNode* slow = head;
       // 快慢指针遍历
       while (true) {
           // fast 遍历到链表尾部
           if (fast == nullptr || fast->next == nullptr)
                return nullptr; 
            fast = fast->next->next;
            slow = slow->next;
            if (fast == slow) 
                break;
       }
       // fast从头开始遍历
       fast = head;
       while (slow != fast) {
           slow = slow->next;
           fast = fast->next;
       }
       return fast;
    }
};
```
