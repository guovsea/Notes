## 内容

```cpp
/**
 * @brief
 *
 * @tparam T
 * @tparam X  传入 策略类（policy class） 或 标签类型（tag type)
 * @tparam N  区分同一类型 T 创建的多个独立的单例实例
 */
template <class T, class X = void, int N = 0> class Singletion {
  public:
    static T *Instance() {
        static T v;
        return &v;
    }
};

template <class T, class X = void, int N = 0> class SingletionSharedPtr {
    public :
    static std::shared_ptr<T> Instance() {
        ståd::shared_ptr<T> v(new T());
        return v;
    }
};
```

X


```cpp
// usage
struct FastSingleton {};  // 标签类型
struct ThreadSafeSingleton {}; // 另一个标签类型

auto instance1 = Singleton<MyClass, FastSingleton>::GetInstance();
auto instance2 = Singleton<MyClass, ThreadSafeSingleton>::GetInstance();
```


N

- `N` 是 **编译期常量**，通常用于区分基于数值的版本或配置。
- 使用 `N` 参数可以让你为同一类型 `T` 创建多个独立的单例实例，而这些实例通过 `N` 的不同值来区分。
- `N` 更适合用于版本控制、配置编号等场景。例如

```cpp
// usage
auto instanceV1 = Singleton<MyClass, void, 1>::GetInstance(); // 版本 1
auto instanceV2 = Singleton<MyClass, void, 2>::GetInstance(); // 版本 2
```
