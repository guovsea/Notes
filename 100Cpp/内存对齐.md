## 什么是内存对齐

结构体中的不同类型的成员并不是紧密排列，而是中间存在 padding 的。

```cpp
struct test {
    int a;        // 0
    char b;       // 4 ，b c 之间存在 3 个字节的填充
    long long c;  // 8
    char d[10];   // 16
    int *e;       // 32
    int f;        // 40
    int g;        // 44
}; 

```

## 面试

对齐规则：

1. 第一个成员在偏移量位 0 的位置
2. 其他成员变量要对齐到对齐数的整数倍的地址处
   内置类型的对齐数 = min(成员大小，默认对齐数)
3. 结构体的大小为所有成员变量中最大对齐数的整数倍，结构体的对齐数为所有成员的最大对齐数
4. 结构体的对齐数不等于结构体的大小
