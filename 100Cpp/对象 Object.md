C++ 程序可以创建、销毁、引用、访问并操作对象。

在 C++ 中，一个对象拥有这些性质：

- 大小（可以使用 sizeof 获取）；
- 对齐要求（可以使用 alignof 获取）；
- [[存储期 Storage Duration]]（自动、静态、动态、线程局部）；
- [[生存期 Lifetime]]（与存储期绑定或者为临时）
- 类型；
- 值（可能是不确定的，例如未初始化的内置类型的对象）；
- 名字（可选，因为还有匿名对象）


以下实体都不是对象：值，引用，函数，枚举项，类型，类的非静态成员，模板，类或函数模板的特化，命名空间，形参包，和 this。

>区别于：变量(variable)——变量是由声明引入的对象或是普通数据成员（不是静态数据成员）的引用。*不理解，不重要*
