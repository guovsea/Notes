## 常用注释风格

```cpp
/// 单行 Doxygen 注释
/// @brief 简要描述

/**
 * @brief 简要描述
 * @param name 参数说明
 * @return 返回值说明
 * @note 注意事项
 * @see 参考
 */
```

## 常用标签

| 标签 | 说明 |
|------|------|
| `@brief` | 简要描述 |
| `@param` | 参数说明 |
| `@return` | 返回值说明 |
| `@note` | 注意事项 |
| `@see` | 交叉引用 |
| `@todo` | 待办事项 |
| `@deprecated` | 已废弃 |
