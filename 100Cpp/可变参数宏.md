## 限制

想在可变参数宏中实现将多个形参转换为字符串是不可行的。

有这样一个函数，想不传入字符串是不可行的。

```cpp
solar::FromYamlString(str, "name", name, "age", age, "sex", sex);
```

## 可变参数宏语法

```cpp
#define MACRO(name, ...) // __VA_ARGS__ 代表可变参数部分
```

`__VA_ARGS__` 在宏展开时被替换为传入的可变参数。

## 常见用法

```cpp
// 日志宏示例
#define LOG(fmt, ...) printf("[LOG] " fmt "\n", __VA_ARGS__)

LOG("value = %d, name = %s", 42, "test");
// 展开为: printf("[LOG] " "value = %d, name = %s" "\n", 42, "test");
```

## 注意事项

- `##__VA_ARGS__`（GNU 扩展 / C++20 `__VA_OPT__`）：当可变参数为空时，去掉前面的逗号
- 宏展开是纯文本替换，没有类型检查
