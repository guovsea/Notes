## 4. 重载决议 （Overload Resolution）

在候选函数集合进行重载决议


```cpp
template<typename T>  
void f(const T& x) {  
  
}  
  
void f(const double& x) {  
  
}  
TEST(A, a) {  
    f(10);  
}
```

实例化模板 `void f(const int&)` 进入了候选函数集合。

但在重载决议的时候，`void f(const double&)` 是最匹配的函数
