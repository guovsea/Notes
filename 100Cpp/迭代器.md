## 概念

迭代器是一种抽象，提供统一的方式遍历容器中的元素，是 STL 算法与容器之间的桥梁。

## 迭代器分类

| 类别 | 支持操作 | 示例容器 |
|------|----------|----------|
| 输入迭代器 | `*it`, `++it`, `==`, `!=` | `istream_iterator` |
| 输出迭代器 | `*it=`, `++it` | `ostream_iterator` |
| 前向迭代器 | 输入 + 输出 + 多次遍历 | `forward_list` |
| 双向迭代器 | 前向 + `--it` | `list`, `set`, `map` |
| 随机访问迭代器 | 双向 + `+n`, `-n`, `[]`, `<` | `vector`, `deque`, `array` |

## 迭代器失效

迭代器失效的本质：迭代器内部持有的指针或索引指向的内存已被释放、移动或重新分配。

常见失效场景：
- **vector**：`insert`/`erase` 可能导致后续迭代器失效；`push_back` 触发扩容时所有迭代器失效
- **list/forward_list**：只有被 `erase` 的元素的迭代器失效
- **deque**：`insert`/`erase` 都会使所有迭代器失效
- **关联容器**：只有被 `erase` 的元素的迭代器失效

> `erase` 返回指向下一个有效元素的迭代器，可以用它安全地继续遍历。
