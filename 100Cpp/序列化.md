## 序列化时读取文件被截断

可能是未以二进制模式打开文件，或者读取长度计算有误。

## 序列化时 `\n` 变成了 `\r\n`

创建 `std::ifstream` / `std::ofstream` 时没有指定格式为二进制格式，默认的文本模式会在 Windows 上将 `\n` 转换为 `\r\n`。

```cpp
std::ofstream os("output.bin", std::ios::binary);
```

> 在 Windows 平台上，文本模式的 I/O 会自动进行换行符转换，而二进制模式不会。
