操作符的优先级和结合性都有明确的定义。

## 优先级

操作符的优先级（precedence）决定两种不同类型的操作符，先计算哪种操作符。

> 例如：先乘除后加减
## 结合性

操作符的结合性（associativity）决定某种操作符有多个相邻时，先计算（使用）哪边的操作符。左结合先使用左边的操作符；右结合先使用右边的操作符。

例如：

```cpp
int x = 10 - 3 - 2;
```

左结合（Left-to-right）

```cpp
int x = (10 - 3) - 2;
```

右结合（Right-to-left）

```cpp
int x = 10 - (3 - 2);
```
