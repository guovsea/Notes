## 定义

每个对象和引用都有生存期，这是一项运行时性质：每个对象或引用在程序执行时都存在一个时刻开始它的生存期，也都存在一个时刻结束它的生存期。

生存期的概念实际上就是 [[存储期 Storage Duration]] + 临时对象的生存期

## 临时对象的生存期

临时对象（如函数返回的右值、表达式中间结果）在**创建它的完整表达式结束时**被销毁。

有两种例外会延长临时对象的生存期：

1. **绑定到 const 引用**：临时对象绑定到 `const T&` 时，生存期延长至引用的生命周期结束
2. **聚合初始化中的临时对象**（C++20 起）：临时对象在初始化列表中绑定到引用成员时，生存期延长

```cpp
const std::string& ref = std::string("hello"); // 临时对象生存期延长至 ref 离开作用域
```
