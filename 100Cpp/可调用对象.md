1. **函数指针**
   
   函数指针是指向函数的指针，可以通过指针调用函数。

2. **函数对象（也称为仿函数)**
   
   函数对象重载了函数调用操作符 **`operator()`**，可以像函数一样调用。

```cpp
struct MyFunctor {
    int operator()(int x, int y) {
        return x + y;
    }
};

MyFunctor functor;
int result = functor(1, 2); // 调用函数对象
```
3. **Lambda 表达式**
   Lambda 表达式是一种方便创建匿名函数对象的方式，可以在需要时定义并立即使用。

4. **成员函数指针**
   **指向成员函数的指针**，可以看做关联了类类型的函数指针

```cpp
class MyClass {
public:
    int add(int x, int y) {
        return x + y;
    }
};
// 和普通指针并无太大差异，只是需要声明作用域
int (MyClass::*mFPointer)(int, int) = &MyClass::add;
MyClass obj;
int result = (obj.*mFPointer)(1, 2); // 调用成员函数
```

5. **函数指针成员**
   函数指针成员是类的成员变量，它是一个指针，指向某个函数
```cpp
int  global_add(int x, int y) {
	return x + y;
}

class MyClass {
public:
	int (*m_global)(int, int);
	int (MyClass::*m_class)(int, int);

	MyClass() {
		m_class = &MyClass::add;
		m_global = global_add;
	}

	int add(int x, int y) {
		return x + y;
	}
};


MyClass obj;
int result = (obj.*obj.m_class)(1, 2); // 调用成员函数, 
             // (obj.*(obj.m_class))(1, 2);
int result2 = obj.m_global(1, 2);

```
   

6. **标准库中的函数对象**
   C++标准库提供了许多预定义的函数对象，如 [[function|std::function]]、[[bind|std::bind]] 、std::mem_fn等，用于支持更多的灵活性和功能。
