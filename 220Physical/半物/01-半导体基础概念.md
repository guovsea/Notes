# 01 - 半导体基础概念

> 本章介绍半导体的基本物理性质，为理解掺杂机制奠定基础。

---

## 晶体结构与周期性

### 晶体的基本特征

半导体材料都是**晶体**，其原子按照严格的周期性排列，形成**晶格**（Lattice）。这种周期性排列赋予了半导体独特的电学和光学性质。

**晶体的三个基本特征：**
1. **周期性**：原子在空间中呈周期性重复排列
2. **对称性**：具有特定的旋转、平移对称操作
3. **各向异性**：不同晶向具有不同的物理性质

### 常见半导体晶体结构

#### 1. 金刚石结构（Diamond Structure）

**代表材料**：硅（Si）、锗（Ge）

**结构特点**：
- 面心立方（FCC）布拉维格子，基元含2个原子
- 每个原子与4个最近邻原子形成**共价键**
- 配位数为4，呈正四面体构型
- 晶格常数：Si为0.543 nm，Ge为0.566 nm

```
        ●
       /|\
      / | \
     ●—●—●
      \ | /
       \|/
        ●

正四面体结构示意（每个原子连接4个最近邻）
```

**立方晶胞中的原子数**：
- 角上原子：8 × 1/8 = 1
- 面心原子：6 × 1/2 = 3
- 内部原子：4
- **总计**：8个原子/晶胞

#### 2. 闪锌矿结构（Zinc Blende Structure）

**代表材料**：砷化镓（GaAs）、磷化铟（InP）

**结构特点**：
- 与金刚石结构相同的几何构型
- 由两种不同原子交替占据格点位置
- Ga原子形成FCC子晶格，As原子形成另一个FCC子晶格
- 两个子晶格沿体对角线方向位移1/4长度套构而成

**与金刚石结构的区别**：
| 特性 | 金刚石结构 | 闪锌矿结构 |
|------|-----------|-----------|
| 组成 | 单一元素（Si, Ge） | 化合物（GaAs, InP） |
| 对称性 | 中心对称 | 非中心对称 |
| 压电效应 | 无 | 有 |
| 极性 | 无极性 | 有极性（[111]方向） |

### 晶向与晶面

**重要晶向**：
- [100]方向：沿立方体棱边，常用于MOSFET沟道
- [110]方向：沿面对角线
- [111]方向：沿体对角线，GaAs生长常用方向

**晶面表示**：用米勒指数（hkl）表示
- (100)面：与x轴垂直的晶面
- (111)面：八面体面，原子密度最高

## 常见半导体材料

### 元素半导体

#### 硅（Silicon, Si）
**地位**：现代微电子工业的基石，95%以上的集成电路使用硅

**基本参数**：
| 参数 | 数值（300K） | 说明 |
|------|-------------|------|
| 原子序数 | 14 | 第IV族元素 |
| 晶格常数 | 0.543 nm | 金刚石结构 |
| 禁带宽度 | 1.12 eV | 间接带隙 |
| 本征载流子浓度 | 1.5 × 10¹⁰ cm⁻³ | 300K |
| 电子迁移率 | 1450 cm²/(V·s) | 低掺杂时 |
| 空穴迁移率 | 500 cm²/(V·s) | 低掺杂时 |
| 介电常数 | 11.9 | 相对介电常数 |

**优点**：
- 储量丰富（地壳中第二丰富元素）
- 氧化生成高质量的SiO₂，是优良的绝缘层
- 工艺成熟，成本低廉
- 机械强度高，易于加工

**缺点**：
- 间接带隙，发光效率低
- 载流子迁移率相对较低

#### 锗（Germanium, Ge）
**地位**：第一个晶体管的材料，现主要用于高速器件和光探测器

**基本参数**：
| 参数 | 数值（300K） | 说明 |
|------|-------------|------|
| 原子序数 | 32 | 第IV族元素 |
| 晶格常数 | 0.566 nm | 金刚石结构 |
| 禁带宽度 | 0.66 eV | 比Si窄 |
| 本征载流子浓度 | 2.4 × 10¹³ cm⁻³ | 300K，比Si高3个数量级 |
| 电子迁移率 | 3900 cm²/(V·s) | 约为Si的2.7倍 |
| 空穴迁移率 | 1900 cm²/(V·s) | 约为Si的3.8倍 |

**特点**：
- 载流子迁移率高，适合高速器件
- 禁带窄，本征载流子浓度高，高温漏电大
- 不能像Si那样形成高质量的自然氧化物

### 化合物半导体

#### 砷化镓（Gallium Arsenide, GaAs）
**地位**：高频、高速、光电子器件的主要材料

**基本参数**：
| 参数 | 数值（300K） | 说明 |
|------|-------------|------|
| 禁带宽度 | 1.42 eV | 直接带隙 |
| 电子迁移率 | 8500 cm²/(V·s) | 约为Si的6倍 |
| 饱和电子速度 | 2 × 10⁷ cm/s | 高场下优势更明显 |

**优势**：
- **直接带隙**：高效发光，用于LED、激光器
- **高电子迁移率**：适合高频器件（>100 GHz）
- **半绝缘衬底**：可直接在衬底上做互连，减小寄生电容

## 共价键与价电子

### 共价键的形成

在硅晶体中，每个硅原子有**4个价电子**，与4个最近邻硅原子各共享1对电子，形成4个**共价键**。

**共价键的特点**：
1. **方向性**：沿正四面体的四个顶点方向
2. **饱和性**：每个原子形成固定数量的共价键
3. **结合力**：量子力学交换相互作用

### 价电子与束缚电子

| 电子类型 | 定义 | 特点 |
|---------|------|------|
| **价电子** | 最外层参与成键的电子 | 能量较高，可被激发 |
| **束缚电子** | 参与共价键的电子 | 在绝对零度时不能自由移动 |
| **内层电子** | 不参与成键的电子 | 能量低，紧密结合于原子核 |

### 共价键的断裂与载流子产生

在**绝对零度（0 K）**时：
- 所有价电子都被束缚在共价键中
- 没有自由移动的载流子
- 半导体表现为**绝缘体**

在**有限温度（T > 0 K）**时：
- 热振动可能使部分价电子获得足够能量
- 电子挣脱共价键束缚，成为**自由电子**
- 原共价键位置留下**空穴**（缺失电子的状态）

```
挣脱前：              挣脱后：
  ●─●                  ●  ○(空穴)
  |                    |
  ●─●                  ●  ●(自由电子)

● = 束缚电子    ○ = 空穴    游离的● = 自由电子
```

## 本征半导体的定义

### 定义

**本征半导体（Intrinsic Semiconductor）**是指：
- 完全纯净、无杂质的半导体
- 晶体结构完整、无缺陷
- 载流子仅由热激发产生

### 本征载流子的产生机制

本征激发是**成对产生**的：
- 每产生一个自由电子，就同时产生一个空穴
- 电子浓度 $n$ = 空穴浓度 $p$ = 本征载流子浓度 $n_i$

$$n = p = n_i$$

**产生率（Generation Rate）**：
$$G = \alpha \cdot n_i^2$$
其中 $\alpha$ 是与温度相关的系数。

**复合率（Recombination Rate）**：
$$R = \beta \cdot n \cdot p$$
热平衡时，$G = R$

---

## 本征载流子浓度

### 理论表达式

本征载流子浓度由费米-狄拉克统计推导得到：

$$n_i = \sqrt{N_c N_v} \exp\left(-\frac{E_g}{2k_BT}\right)$$

其中：
- $N_c$：导带有效态密度（Effective Density of States in Conduction Band）
- $N_v$：价带有效态密度（Effective Density of States in Valence Band）
- $E_g$：禁带宽度（Band Gap Energy）
- $k_B$：玻尔兹曼常数（8.617 × 10⁻⁵ eV/K）
- $T$：绝对温度（K）

### 有效态密度

**导带有效态密度**：
$$N_c = 2\left(\frac{2\pi m_n^* k_BT}{h^2}\right)^{3/2} = 2.5 \times 10^{19} \left(\frac{m_n^*}{m_0}\right)^{3/2} \left(\frac{T}{300K}\right)^{3/2} \text{cm}^{-3}$$

**价带有效态密度**：
$$N_v = 2\left(\frac{2\pi m_p^* k_BT}{h^2}\right)^{3/2} = 2.5 \times 10^{19} \left(\frac{m_p^*}{m_0}\right)^{3/2} \left(\frac{T}{300K}\right)^{3/2} \text{cm}^{-3}$$

其中 $m_n^*$ 和 $m_p^*$ 分别为电子和空穴的**有效质量**，$m_0$ 为自由电子质量。

### 温度依赖性

本征载流子浓度强烈依赖于温度：

$$n_i \propto T^{3/2} \exp\left(-\frac{E_g}{2k_BT}\right)$$

**硅的本征载流子浓度**：
| 温度（K） | n_i（cm⁻³） |
|----------|------------|
| 300 | 1.5 × 10¹⁰ |
| 400 | 1.1 × 10¹² |
| 500 | 2.2 × 10¹³ |
| 600 | 1.3 × 10¹⁴ |

**重要结论**：
- 温度每升高约10°C，$n_i$ 大约翻倍
- 高温下半导体趋向本征化（载流子主要由本征激发产生）
- 这是半导体器件高温性能劣化的主要原因

## 载流子的产生与复合

### 产生机制

**1. 本征产生（热产生）**
- 价带电子获得热能跃迁到导带
- 需要能量 ≥ 禁带宽度 $E_g$
- 产生电子-空穴对

**2. 光产生**
- 光子能量 $h\nu \geq E_g$ 时，可被吸收
- 产生额外的电子-空穴对
- 光电导效应的基础

### 复合机制

**1. 直接复合（带间复合）**
- 导带电子直接跃迁到价带，与空穴复合
- 辐射出光子（发光）或声子（发热）
- 在直接带隙半导体中占主导

**2. 间接复合（通过复合中心）**
- 通过禁带中的杂质或缺陷能级进行
- 分为三个步骤：电子俘获、空穴俘获
- 在间接带隙半导体（如Si）中占主导

**3. 俄歇复合（Auger Recombination）**
- 电子-空穴复合时，能量传递给第三个载流子
- 在高载流子浓度时变得重要
- 不涉及光子发射

### 载流子寿命

**定义**：非平衡载流子从产生到复合的平均存在时间。

$$\tau = \frac{1}{\text{复合概率}}$$

对于直接复合：
$$\tau = \frac{1}{\beta \cdot n_i}$$

对于间接复合：
$$\tau = \frac{1}{\sigma \cdot v_{th} \cdot N_t}$$

其中：
- $\sigma$：俘获截面
- $v_{th}$：载流子热运动速度
- $N_t$：复合中心浓度

**典型值**：
- 高质量Si单晶：τ ~ 毫秒量级
- 一般器件级Si：τ ~ 微秒量级
