## 相关

- [[124. 二叉树中的最大路径和]]

给定一个二叉树的根节点 `root` ，和一个整数 `targetSum` ，求该二叉树里节点值之和等于 `targetSum` 的 **路径** 的数目。

**路径** 不需要从根节点开始，也不需要在叶子节点结束，但是路径方向必须是向下的（只能从父节点到子节点）。

**示例 1：**

![](https://assets.leetcode.com/uploads/2021/04/09/pathsum3-1-tree.jpg)

**输入：** root = [10,5,-3,3,2,null,11,3,-2,null,1], targetSum = 8
**输出：** 3
**解释：** 和等于 8 的路径有 3 条，如图所示。

**示例 2：**

**输入：** root = [5,4,8,11,null,13,4,7,2,null,null,5,1], targetSum = 22
**输出：** 3

**提示:**

- 二叉树的节点个数的范围是 `[0,1000]`
- `-109 <= Node.val <= 109` 
- `-1000 <= targetSum <= 1000`

## 解答

**方法 1 深度优先搜索**
```cpp
class Solution {
public:
    int pathSum(TreeNode* root, int targetSum) {
        vector<long> sumList;
        int ways = 0;
        process(root, targetSum, sumList, ways);
        return ways;
    }
    void process(TreeNode* root, int targetSum, vector<long>& sumList, int& ways) {
        if (root == nullptr) {
            return;
        }
        // 给前面所有路径和都加上当前节点的val
        for(auto &sum : sumList) {
            sum += root->val;
        }
        // 将当前节点的val添加进路径和列表
        sumList.push_back(root->val);
        for (auto sum : sumList) {
            if (sum == targetSum) {
                // ways保存路径总数
                ways += 1;
            }
        }
        process(root->left, targetSum, sumList, ways);
        process(root->right, targetSum, sumList, ways);
    }
};
```

**复杂度分析（方法 1）**

- 时间复杂度：O(n^2)，每个节点维护一个路径和列表
- 空间复杂度：O(n)，递归栈深度

同 [[124. 二叉树中的最大路径和]] 一样，都是以节点为作为子问题去思考。子问题是**以 root 为起点向下且满足路径综合为 targetSum 的路径数目**。遍历所有节点求和便能得到整棵树的路径和。

**二叉树一般都以所要求解的问题为子问题，如果需要转换，那么就会是难题**
