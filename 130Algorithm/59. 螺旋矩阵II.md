**本题并不涉及到什么算法，就是模拟过程，但却十分考察对代码的掌控能力**

一定要坚持 [[循环不变量原则]]。

而求解本题依然是要坚持循环不变量原则。

模拟顺时针画矩阵的过程:

- 填充上行从左到右
- 填充右列从上到下
- 填充下行从右到左
- 填充左列从下到上

**每画一条边都要坚持一致的左闭右开，或者左开右闭的原则，这样这一圈才能按照统一的规则画下来。**


## 要点

- 注意需要哪些变量，可以先确定画圈次数，然后再想每个圈需要的参数
- 注意 `res[i][j]` 按行案列时改变的变量是哪个，j 改变是往左或往右；i 改变是往上或往下
- 往左和往上遍历时，终点不是 `0 + 1`，而是 `startx/starty + 1`，一个一个圈地缩小


```cpp
class Solution {
public:
    vector<vector<int>> generateMatrix(int n) {
        vector<vector<int>> res(n, vector<int>(n, 0)); // 使用vector定义一个二维数组
        int startx = 0, starty = 0; // 定义每个圈的起始位置
        int loop = n /
         2; // 需要画多少个圈
        int mid = n / 2; // 矩阵中间的位置，例如：n为3， 中间的位置就是(1，1)，n为5，中间位置为(2, 2)
        int v = 1; // 用来给矩阵中每一个空格赋值
        int offset = 1; // 需要控制每一条边遍历的长度，每次画圈后右边界收缩一位
        int i,j;

        while (loop > 0) {
            i = startx;
            j = starty;
            for (; j < n - offset; ++j) {
                res[i][j] = v++;
            }
            for (; i < n - offset; ++i) {
                res[i][j] = v++;
            }
            for (; j > startx; --j) {
                res[i][j] = v++;
            }
            for (; i > starty; --i) {
                res[i][j] = v++;
            }
            startx++;
            starty++;
            loop--;
            offset++;
        }
        // ! 如果 n 为 奇数，中间会剩下1格，需要单独为它赋值
        if (n % 2 == 1) {
            res[n/2][n/2] = v;
        }
        return res;
    }
};
```
**复杂度分析**

- 时间复杂度：O(n^2)，需要填充 n*n 个元素
- 空间复杂度：O(n^2)，结果矩阵
