## 解答

```cpp
class Solution {
public:
    ListNode* deleteDuplicates(ListNode* head) {
        if (head == nullptr || head->next == nullptr) {
            return head;
        }
        ListNode dummy(101);
        dummy.next = head;
        ListNode* tail = &dummy; // 记录最后返回的链表的尾部指针，后续将结点添加在它后面
        ListNode* last = head;
        while (last != nullptr) {
            // 记录取值为x的第一个节点
            ListNode* first = last;
            // 如果下一个节点和当前节点重复，那就遍历到下一个不是该值的节点
            while (last->next != nullptr && last->val == last->next->val) {
                last = last->next;
            }
            // 如果取值为x的第一个节点等于取值为x的最后一个节点，说明只有一个x
            if (first == last) {
                tail->next = first; 
                tail = tail->next;
            }
            last = last->next;
        }
        tail->next = nullptr; // 处理last为最后一个节点的情况 eg:[1, 1]
        return dummy.next;  
    }
};
```

**关键**

用两个节点`first`和`last`分别保存第一个等于`x`和最后一个等于`x`的节点，如果这两个节点相等，那么就只有一个`x`，因此将该节点放到`tail`后面。**如果不相等，则跳过所有重复节点**。最后需要 `tail->next = nullptr` 处理尾部。
## 题目

给定一个已排序的链表的头 `head`，删除原始链表中所有重复数字的节点，只留下不同的数字 。返回 _已排序的链表_ 。

**示例 1：** *注：重复数字的所有节点都要删除*

![](https://assets.leetcode.com/uploads/2021/01/04/linkedlist1.jpg)

**输入：** head = [1,2,3,3,4,4,5]

**输出：** [1,2,5]

**示例 2：**

![](https://assets.leetcode.com/uploads/2021/01/04/linkedlist2.jpg)

**输入：** head = [1,1,1,2,3]
**输出：**[2,3]

**提示：**

- 链表中节点数目在范围 `[0, 300]` 内
- `-100 <= Node.val <= 100`
- 题目数据保证链表已经按升序 **排列**
