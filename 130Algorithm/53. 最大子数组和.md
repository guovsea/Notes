## 解答

```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int maxSum = -1e9;
        int currMaxSum = -1e9;
        for (int i = 0; i < nums.size(); i++) {
            // 要么将nums[i]加入到所考虑的连续数组中，要么从nums[i]重新开始
            currMaxSum = max(currMaxSum + nums[i], nums[i]);
            // 记录答案
            maxSum = max(maxSum, currMaxSum);
        }
        return maxSum;
    }
};
```
## 题目

[力扣（LeetCode）官网 - 全球极客挚爱的技术成长平台](https://leetcode.cn/problems/maximum-subarray/description/)

给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**子数组** 是数组中的一个连续部分
**复杂度分析**

- 时间复杂度：O(n)
- 空间复杂度：O(1)

**子问题**：`dp[i]` 以 `nums[i]` 结尾的最大子数组和。状态转移：`dp[i] = max(dp[i-1] + nums[i], nums[i])`
