访问链表中第 k 个元素

1. **p 指向头结点，遍历 k-1 次**
```cpp
    ListNode* p = head; 
    for (int i = 0; i < k - 1; i++) {
        p = p->next;
    }
```
2. **p 指向 dummy 节点，遍历 k 次**
```cpp
    ListNode* p = dummy; 
    for (int i = 0; i < k; i++) {
        p = p->next;
    }
```

访问链表中倒数第 k 个元素

 **first 和 secode 都指向 head，first 先走 k 步**
```cpp
    ListNode *second = head, *first = head;
    for (int i = 0; i < k; i++) {
        first = first->next;
    }
    while (first != nullptr) {
        first = first->next;
        second = second->next;
    }
    cout << "second-> " << second->val << endl;
```
