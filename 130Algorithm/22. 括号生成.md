## 解答

回溯法，以左括号剩余数量和右括号剩余数量作为状态变量。

**剪枝条件**：右括号剩余数量小于左括号剩余数量时，说明已经使用了过多右括号，不合法，直接返回。

```cpp
class Solution {
public:
    vector<string> generateParenthesis(int n) {
        vector<string> res;
        dfs("", n, n, res);
        return res;
    }
    // left: 剩余左括号数量, right: 剩余右括号数量
    void dfs(string path, int left, int right, vector<string>& res) {
        if (left == 0 && right == 0) {
            res.push_back(path);
            return;
        }
        // 剪枝：右括号剩余数量不能小于左括号剩余数量
        if (right < left) return;
        if (left > 0) {
            dfs(path + "(", left - 1, right, res);
        }
        if (right > 0) {
            dfs(path + ")", left, right - 1, res);
        }
    }
};
```

**复杂度分析**

- 时间复杂度：O(4^n / sqrt(n))，卡特兰数
- 空间复杂度：O(n)，递归栈深度

## 题目

数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。

**示例 1：**

**输入：** n = 3
**输出：** ["((()))","(()())","(())()","()(())","()()()"]

**示例 2：**

**输入：** n = 1
**输出：** ["()"]
