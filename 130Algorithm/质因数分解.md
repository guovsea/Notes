
## 一、问题描述

给定一个正整数 $n$，求出 $n$ 的所有质因数（含重复因子），以列表形式返回。

| 输入 | 输出 | 解释 |
|------|------|------|
| `n = 18` | `[2, 3, 3]` | $18 = 2 \times 3^2$ |
| `n = 25` | `[5, 5]` | $25 = 5^2$ |
| `n = 13` | `[13]` | 13 本身是质数 |

---

## 二、核心数学原理

### 定理：每个合数至少有一个质因数 ≤ $\sqrt{n}$

**反证法：**

设 $a$、$b$ 是 $n$ 的两个因数，$a \times b = n$。

若两者都 $> \sqrt{n}$，则：
$$a \times b > \sqrt{n} \times \sqrt{n} = n$$

这与 $a \times b = n$ 矛盾。

因此，**至少有一个因数 ≤ $\sqrt{n}$**，只需枚举到 $\sqrt{n}$ 即可找到所有质因数。

---

## 三、思路转化：从问题到代码

### 3.1 目标操作

把 $n$ 反复除以质数，直到 $n = 1$。每除一次，记录那个质数。例如：

$$360 = \underbrace{2 \times 2 \times 2}_{除以2三次} \times \underbrace{3 \times 3}_{除以3两次} \times \underbrace{5}_{除以5一次}$$

### 3.2 关键问题：怎么保证除到的一定是质数？

朴素想法是：先筛出所有质数，再逐一尝试。但**我们根本不需要筛**。

**核心论证如下：**

> **前提**：我们从小到大依次枚举 $i = 2, 3, 4, 5, 6, \ldots$，对每个 $i$ 都**完全除尽**后才推进到下一个。

当枚举到某个 $i$ 时，$n$ 的当前值已经被所有 $< i$ 的因子**彻底除干净了**。

现在假设 $i$ 是**合数**，那么 $i$ 一定能分解为若干个质因数：
$$i = p_1 \times p_2 \times \cdots \times p_k$$
其中每个 $p_j < i$。

但这些 $p_j$ 在之前枚举时已经**被完全除尽**了，意味着当前 $n$ 中不再包含任何 $p_j$。

既然 $n$ 中不包含 $p_1, p_2, \ldots, p_k$ 中的任何一个，那 $n$ 自然也不包含它们的乘积 $i$。

**所以：$n$ 一定不被 $i$ 整除，合数自动被跳过。**

### 3.3 一个具体的例子

$n = 360 = 2^3 \times 3^2 \times 5$：

| 枚举 $i$ | 质还是合 | $n$ 变化 | 除得动吗 |
|:---------:|:-------:|:--------:|:-------:|
| 2 | 质 | 360→180→90→**45** | ✅ 除 3 次 |
| 3 | 质 | 45→15→**5** | ✅ 除 2 次 |
| **4** | **合** (2×2) | 5 | ❌ 除不动 |
| 5 | 质 | 5→**1** | ✅ 除 1 次 |
| **6** | **合** (2×3) | 1 | ❌ 除不动 |
| 停止 | $i^2 > n$ | — | — |

> 合数 4 和 6 完全除不动——因为它们的质因子 2、3 早被剥离干净了。

### 3.4 一句话总结

> **从小到大枚举 + 每个数完全除尽，保证了任何合数在被枚举到之前，其所有质因子已从 $n$ 中移除，因此合数天然除不动，无需额外过滤。**

> 以 6k±1 步长枚举质数的策略: [[质数判断（试除法 + 6k±1 优化）]] 
## 四、代码实现

```cpp
std::vector<int> primeFactor(int n) {
    std::vector<int> rt;

    // 提取所有因子 2
    while (n % 2 == 0) {
        n /= 2;
        rt.push_back(2);
    }

    // 提取所有因子 3
    while (n % 3 == 0) {
        n /= 3;
        rt.push_back(3);
    }

    // 从 5 开始，以 6k±1 步长枚举
    for (int i = 5; i * i <= n; i += 6) {
        while (n % i == 0) {
            n /= i;
            rt.push_back(i);
        }
        while (n % (i + 2) == 0) {
            n /= (i + 2);
            rt.push_back(i + 2);
        }
    }

    // 若剩余 n > 1，则 n 本身是质因数
    if (n > 1) {
        rt.push_back(n);
    }

    return rt;
}
```

### 易错点速查

| 要点                 | 说明                                |
| ------------------ | --------------------------------- |
| 用 `while` 不是 `if`  | 同一个质因子可能出现多次（如 $25 = 5^2$）        |
| 循环条件用 `i * i <= n` | 避免 `sqrt()` 的浮点精度问题               |
| 最后 `if (n > 1)`    | 循环结束后若 $n > 1$，说明 $n$ 本身就是最后一个质因数 |
| $n = 1$ 时返回空       | 1 没有质因数，代码自然返回空列表                 |
