# 递归中的入栈顺序与出栈顺序

---

## 一、核心结论

> **递归调用之前**的代码 → 按入栈顺序执行（从大到小，即 `n → 1`）
> **递归调用之后**的代码 → 按出栈顺序执行（从小到大，即 `1 → n`）

---

## 二、通用模板

```cpp
void recursion(int n) {
    if (n == 0) return;           // 基线条件（栈底）

    // ══════ 入栈阶段 ══════
    // 这里的代码按 n, n-1, n-2, ... 1 的顺序执行
    cout << "入栈: " << n << endl;

    recursion(n - 1);             // 🔑 递归调用

    // ══════ 出栈阶段 ══════
    // 这里的代码按 1, 2, 3, ... n 的顺序执行
    cout << "出栈: " << n << endl;
}
```

## 三、Print 1 to N（递归调用**之后**输出）

```cpp
#include <iostream>
#include <vector>
using namespace std;

void print1toN(int n) {
    if (n == 0) return;
    print1toN(n - 1);    // 先递归（入栈）
    cout << n << " ";    // 后打印（出栈时执行）
}

int main() {
    print1toN(5);
    // 输出: 1 2 3 4 5
    return 0;
}
```

### 调用栈过程（n = 3）

```
入栈 ↓                          出栈 ↑

print1toN(3)
 └─ print1toN(2)
     └─ print1toN(1)
         └─ print1toN(0)  ←── 触底，开始返回
         └─ return
     ←── 回到 print1toN(1) ──→ print(1) ✓
 ←── 回到 print1toN(2) ──→ print(2) ✓
←── 回到 print1toN(3) ──→ print(3) ✓
```

## 四、Print N to 1（递归调用**之前**输出）

```cpp
#include <iostream>
using namespace std;

void printNto1(int n) {
    if (n == 0) return;
    cout << n << " ";    // 先打印（入栈时执行）
    printNto1(n - 1);    // 后递归（入栈）
}

int main() {
    printNto1(5);
    // 输出: 5 4 3 2 1
    return 0;
}
```

### 调用栈过程（n = 3）

```
入栈 ↓（打印在这里发生）

printNto1(3) ──→ print(3) ✓
 └─ printNto1(2) ──→ print(2) ✓
     └─ printNto1(1) ──→ print(1) ✓
         └─ printNto1(0)  ←── 触底
         └─ return       （出栈过程中无操作）
```

---

## 五、两者对比

| 对比项 | Print 1 to N | Print N to 1 |
|--------|-------------|-------------|
| 输出语句位置 | 递归调用**之后** | 递归调用**之前** |
| 执行时机 | 📤 **出栈时** | 📥 **入栈时** |
| 执行顺序 | 1 → 2 → … → n | n → n-1 → … → 1 |
| 本质 | 先深入再回头处理 | 边深入边处理 |

## 六、验证：两处都写输出

```cpp
#include <iostream>
using namespace std;

void both(int n) {
    if (n == 0) return;
    cout << "入栈: " << n << endl;   // 入栈阶段
    both(n - 1);
    cout << "出栈: " << n << endl;   // 出栈阶段
}

int main() {
    both(3);
    return 0;
}
```

```
入栈: 3
入栈: 2
入栈: 1
出栈: 1
出栈: 2
出栈: 3
```

> 可以看出：入栈阶段是 `3→2→1`（递减），出栈阶段是 `1→2→3`（递增），完全对称。

## 七、总结记忆口诀

```
递归前 → 入栈序 → 先大后小（n → 1）
递归后 → 出栈序 → 先小后大（1 → n）
基线条件 → 栈底 → 转折点
```

> 💡 **延伸应用**：这个规律在二叉树遍历中体现为——
> - **前序遍历**（入栈序）：根 → 左 → 右
> - **中序遍历**（左子树出栈后）：左 → 根 → 右
> - **后序遍历**（全部出栈后）：左 → 右 → 根
