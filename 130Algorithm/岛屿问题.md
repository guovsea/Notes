## 核心思路

岛屿问题的通用框架：**遍历二维网格，遇到岛屿（'1'）时，用 DFS/BFS 将与之相连的所有陆地标记为已访问，计数加一。**

## 通用 DFS 模板

```cpp
void dfs(vector<vector<char>>& grid, int r, int c) {
    // 边界检查
    if (r < 0 || r >= grid.size() || c < 0 || c >= grid[0].size()) {
        return;
    }
    // 如果不是陆地或已访问，返回
    if (grid[r][c] != '1') {
        return;
    }
    // 标记为已访问（将 '1' 改为 '2' 或 '0'）
    grid[r][c] = '2';
    // 向四个方向扩展
    dfs(grid, r - 1, c);  // 上
    dfs(grid, r + 1, c);  // 下
    dfs(grid, r, c - 1);  // 左
    dfs(grid, r, c + 1);  // 右
}
```

## 岛屿数量

```cpp
class Solution {
public:
    int numIslands(vector<vector<char>>& grid) {
        int count = 0;
        for (int r = 0; r < grid.size(); r++) {
            for (int c = 0; c < grid[0].size(); c++) {
                if (grid[r][c] == '1') {
                    count++;
                    dfs(grid, r, c);
                }
            }
        }
        return count;
    }
};
```

**复杂度分析**

- 时间复杂度：O(m * n)，每个格子最多访问一次
- 空间复杂度：O(m * n)，最坏情况下递归栈深度

## 常见岛屿类变体

| 题目 | 说明 |
|------|------|
| 岛屿数量 | 统计 '1' 连通块的数量 |
| 岛屿最大面积 | DFS 时返回面积 |
| 封闭岛屿数量 | 排除边界上的岛屿 |
| 飞地的数量 | 不与边界相连的陆地格子数 |

**关键点**：所有岛屿问题的核心都是 DFS/BFS 遍历二维网格，区别仅在于统计的内容不同。
