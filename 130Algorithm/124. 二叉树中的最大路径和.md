二叉树中的 **路径** 被定义为一条节点序列，序列中每对相邻节点之间都存在一条边。同一个节点在一条路径序列中 **至多出现一次** 。该路径 **至少包含一个** 节点，且不一定经过根节点。

**路径和** 是路径中各节点值的总和。

给你一个二叉树的根节点 `root` ，返回其 **最大路径和** 。

**示例 1：**

![](https://assets.leetcode.com/uploads/2020/10/13/exx1.jpg)



**输入：** root = [1,2,3]
**输出：** 6
**解释：** 最优路径是 2 -> 1 -> 3 ，路径和为 2 + 1 + 3 = 6

**示例 2：**

![](https://assets.leetcode.com/uploads/2020/10/13/exx2.jpg)

**输入：** root = [-10,9,20,null,null,15,7]
**输出：** 42
**解释：** 最优路径是 15 -> 20 -> 7 ，路径和为 15 + 20 + 7 = 42

**提示：**

- 树中节点数目范围是 `[1, 3 * 104]`
- `-1000 <= Node.val <= 1000`

## 解答

```cpp
class Solution {
public:
    int maxPathSum(TreeNode* root) {
        int res = -1e9;
        pathSum(root, res);
        return res;
    }
    int pathSum(TreeNode* root, int& res) {
        // 包括root节点的序列的最大路径和,不是这个含义！！！！！！
        // 求解的问题是：当前节点为子节点是的贡献值是多少
        // 题目要求至多出现一次，因此当前节点为子节点时，要么向左走，要么向右走
        if (root == nullptr) {
            return 0;
        }
        // 只有两边的路径和>0才会考虑
        int left = max(pathSum(root->left, res), 0);
        int right = max(pathSum(root->right, res), 0);
        int maxPath = root->val + left + right;
        res = max(res, maxPath);
        // return maxPath; // 以当前节点为子节点，到当前节点后要么向左走，要么向右走
        return root->val + max(left, right);
    }
};
```
有一种二叉树题型，要以节点作为子问题去思考，间接得到问题的答案。

该问题中，每个节点对应的问题是：**序列中包含当前节点的最大路径和**， 遍历整个二叉树就能得到所有可能的序列的最大路径和的最大值，也就是整个树的最大路径和。

这道题还需要注意的问题是：子问题解决并返回的问题和所需要求解的问题性质不同。*子问题* 是**以当前节点为子节点**的最大路径和；而 *求解的问题* 包含当前节点的最大路径和。原因是因为题目中：同一个节点在一条路径序列中 **至多出现一次** ，如果即包含一个节点的左子树，有包含它的右子树，那么它就被包含了两次。
