双指针法（快慢指针法）： **通过一个快指针和慢指针在一个 for 循环下完成两个 for 循环的工作。**

定义快慢指针

- 快指针：寻找新数组的元素 ，新数组就是不含有目标元素的数组
- 慢指针：指向更新 新数组下标的位置

## 解答

```cpp
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int slow = 0;
        for (int fast = 0; fast < (int)nums.size(); fast++) {
            if (nums[fast] != val) {
                nums[slow++] = nums[fast];
            }
        }
        return slow;
    }
};
```

**复杂度分析**

- 时间复杂度：O(n)
- 空间复杂度：O(1)
