## 括号生成

[括号生成](https://leetcode.cn/problems/generate-parentheses/)

数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。

有效的括号组合：

方向一：判断已有的括号组合是否有效

栈

方向二：产生有效的括号组合

**dfs，生成子节点的时候确保：添加左括号时保证左括号数小于需要的括号组数，添加右括号时保证右括号数小于左括号**即可生成有效的括号组合。

```text
                 NULL
        ) [left == right, 不能添加有括号]
                 (
(   [left < right]
                     )   ...
        ( [left < right]

```

```cpp
/**
 * @brief 生成有效括号组合
 *
 * @param path     保存 dfs 路径
 * @param n        产生 n 组 括号
 * @param depth    当前 dfs 的深度
 * @param open     左括号的数量
 * @param close    右括号的数量
 * @param res      存储组合
 */
void dfs(string &path, int n, int depth, int open, int close,
         vector<string> &res) {
    if (depth >= 2 * n) {
        res.push_back(path);
        return;
    }
    if (open < n) {
        path.push_back('(');
        dfs(path, n, depth + 1, open + 1, close, res);
        path.pop_back(); // 回溯
    }
    if (close < open) { // 只要保证添加)的时候都能保证，就能够匹配
        path.push_back(')');
        dfs(path, n, depth + 1, open, close + 1, res);
        path.pop_back();
    }
}
```
