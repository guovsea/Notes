## 普通变量

普通的局部变量就是一个列表，用 `;` 分隔。初始化一个局部变量有多种方式：

```CMake
# 1. 用多个字符串
set(FRUIT "apple" "banana" "oriange")
# set(FRUIT "apple" "banana" "oriange;strewberry") # 每个字符串中也能用 ; 分割
message("${FRUIT}")

# 解析命令行参数
separate_arguments(args UNIX_COMMAND PROGRAM "cc -c main.c")
message("args: ${args}")
```

`separate_arguments` 将命令行参数分析到以分号分隔的列表中

## 缓存变量

Cache 变量允许运行 cmake 时在命令行或 GUI 中指定变量的值，并将这些值保存在 CMakeCache.txt 中，以便在后续的 CMake ` 运行中被重用。

```CMake
# type 和 description 都是必须的, description 会在 GUI 界面中作为描述信息
set(VAR "default value" CACHE STRING "Description")
```

在命令行中定义 `CACHE_VAR` 的值：

```shell
cmake -Bbuild -S . -DVAR="hello world"
```

> 命令行给变量赋值时区分变量的大小写

## Properties

`Properties` 类似变量，但是和某种元素相关联（比如 `directory`、`target`、`source file`）

设置 `Properties`

```CMake
set_property(TARGET TargetName
             PROPERTY PropertyName Value)

# 设置 C++ 版本
set_target_properties(TargetName PROPERTIES
                      CXX_STANDARD 11)
# 设置编译标志
set_property(TARGET TargetName 
             PROPERTY COMPILE_FLAGS "-Wall")
# 设置链接标志
set_property(TARGET TargetName 
             PROPERTY LINK_FLAGS "-L/usr/local/lib")
```

获取 `Properties`

```CMake
get_property(ResultVariable TARGET TargetName PROPERTY CXX_STANDARD)
```

## 使用变量

注意事项：

对于一个变量 `VAR`

- `"${VAR}"` 该表达式为一个用 `;` 分割每个元素的字符串
- `${VAR}` 该表达式为一个列表

```CMake
"${VAR}" # 
${VAR}
```
