# 06 - pn结与掺杂应用

> 本章讲解pn结的形成原理、能带结构和电学特性，以及掺杂工程在现代半导体器件中的应用。

---

## pn结的形成过程

### 什么是pn结？

**pn结（pn Junction）**：在同一半导体单晶中，通过掺杂形成的n型区域和p型区域之间的界面过渡区。

pn结是几乎所有半导体器件的基础结构：
- 二极管的核心
- 双极型晶体管（BJT）的基础
- MOS晶体管的源漏结
- 太阳能电池的核心

### 形成pn结的方法

**1. 合金法（早期工艺）**
- 将铝箔放在n型锗片上加热
- 铝溶解入锗形成p区
- 现已淘汰

**2. 扩散法**
- 在n型硅表面扩散硼（或反之）
- 形成缓变结
- 适用于功率器件

**3. 离子注入法**
- 选择性注入形成浅结
- 现代CMOS的主流工艺
- 可精确控制结深和浓度

**4. 外延法**
- 在衬底上外延生长相反类型的层
- 适用于高频器件
- 可实现突变结

## 空间电荷区

### 热平衡时的载流子分布

当n型和p型半导体接触时：

**初始状态**：
- n区：大量电子（$\sim N_D$），空穴很少
- p区：大量空穴（$\sim N_A$），电子很少

**载流子扩散**：
- n区电子向p区扩散
- p区空穴向n区扩散

### 空间电荷区的形成

**扩散过程**：
1. n区电子扩散到p区，与空穴复合
2. p区空穴扩散到n区，与电子复合
3. n区留下带正电的施主离子（$N_D^+$）
4. p区留下带负电的受主离子（$N_A^-$）

**空间电荷区的建立**：
- 界面附近形成电荷区
- n区侧带正电（$\rho = +qN_D$）
- p区侧带负电（$\rho = -qN_A$）

```
p区                    n区
++++++ ─┬─ ────── ++++++  空穴（多子）
        │
─────○○○│●●●●●───────    电子（少子）
        │
○○○○    │    ●●●●        空间电荷区
    ○○○○│●●●
        │
   [Na⁻]│[Nd⁺]           电离杂质
        │
    ○ = 空穴    ● = 电子
```

### 内建电场的建立

空间电荷区产生的电场：
- 方向：从n区指向p区
- 作用：阻止多子继续扩散
- 称为**内建电场**（Built-in Electric Field）

### 耗尽层近似

**假设**：
- 空间电荷区内载流子完全耗尽
- 电荷仅由电离杂质贡献
- 区外为中性区

**泊松方程**：
$$\frac{d^2\phi}{dx^2} = -\frac{\rho}{\epsilon_s}$$

其中：
- n区（$0 < x < x_n$）：$\rho = +qN_D$
- p区（$-x_p < x < 0$）：$\rho = -qN_A$

## 内建电势

### 内建电势的物理意义

**定义**：热平衡时，pn结两侧n区和p区的电势差。

**物理来源**：
- n区费米能级高（靠近导带）
- p区费米能级低（靠近价带）
- 接触后费米能级统一
- 能带弯曲产生电势差

### 内建电势的计算

**从能带角度**：
$$V_{bi} = \frac{E_{Fn} - E_{Fp}}{q}$$

**用掺杂浓度表示**：
$$V_{bi} = \frac{k_BT}{q} \ln\left(\frac{N_A N_D}{n_i^2}\right)$$

**数值示例**（Si, 300K）：
- $N_A = 10^{16}$ cm⁻³
- $N_D = 10^{15}$ cm⁻³
- $n_i = 1.5 \times 10^{10}$ cm⁻³

$$V_{bi} = 0.026 \times \ln\left(\frac{10^{31}}{2.25 \times 10^{20}}\right) = 0.026 \times \ln(4.4 \times 10^{10}) \approx 0.67 \text{ V}$$

### 内建电势的影响因素

| 因素 | 影响 | 说明 |
|------|------|------|
| 掺杂浓度 | $V_{bi} \propto \ln(N_A N_D)$ | 掺杂越高，$V_{bi}$越大 |
| 温度 | $V_{bi}$随T升高而降低 | 温度系数约-2 mV/K |
| 禁带宽度 | 宽禁带材料$V_{bi}$大 | SiC的$V_{bi}$约2-3V |

## pn结的能带图

### 平衡态能带图

**绘制步骤**：
1. 分开画n区和p区的能带
2. 使费米能级对齐（热平衡条件）
3. 能带发生弯曲

```
n区                    p区

E_c ─────────┐         ┌────────────
             │         │
             │    E_F ─┼────────────
             │         │
E_F ─────────┤         │
             │         │
             │         │
E_v ─────────┘         └────────────

        ────→ E  内建电场方向

接触后：

E_c ───────────────┐
                   │
              E_F ─┼───────────────
                   │
                   │
                   └─────────────── E_v
                           E_v
```

**能带弯曲的原因**：
- n区能带相对p区降低$qV_{bi}$
- 电子势能：$-q\phi(x)$
- 保持费米能级恒定

### 正向偏置

**正向偏置**：p区接正，n区接负

**效果**：
- 外电场与内建电场相反
- 势垒降低：$V_{bi} - V_F$
- 耗尽层变窄
- 大量载流子注入

**能带图变化**：
```
平衡态：                    正向偏置VF：

E_c ────────┐               E_c ────┐
            │                       │
E_F ────────┼─────              E_F ├────┐
            │                       │    │
            │                       │    │ qVF
            └─────── E_v            └────┘──── E_v
```

**电流机制**：
- n区电子注入p区（成为p区的少数载流子）
- p区空穴注入n区（成为n区的少数载流子）
- 少子扩散 → 复合 → 形成电流

**电流公式**（Shockley方程）：
$$J = J_s \left[\exp\left(\frac{qV}{k_BT}\right) - 1\right]$$

其中饱和电流密度：
$$J_s = qn_i^2 \left(\frac{D_n}{L_n N_A} + \frac{D_p}{L_p N_D}\right)$$

### 反向偏置

**反向偏置**：p区接负，n区接正

**效果**：
- 外电场与内建电场同向
- 势垒升高：$V_{bi} + V_R$
- 耗尽层展宽
- 只有少子漂移形成微小电流

**能带图变化**：
```
平衡态：                    反向偏置VR：

E_c ────────┐                    ┌─── E_c
            │               ┌────┘
E_F ────────┼─────          ├────┤ E_F
            │               │    │
            │          qVR  │    │
            └─────── E_v    └────┴─── E_v
```

**电流机制**：
- 多子被电场拉离耗尽层
- 只有热产生的少子被电场扫过耗尽层
- 电流很小且饱和（与电压无关）

## 掺杂浓度设计

### 突变结与缓变结

**突变结（Abrupt Junction）**：
- 掺杂在界面处突变
- 电荷分布：矩形
- 常见于离子注入结、外延结

**缓变结（Graded Junction）**：
- 掺杂浓度渐变
- 电荷分布：斜坡
- 常见于扩散结

**线性缓变结**：
$$N_D - N_A = ax$$
其中$a$为浓度梯度。

### 耗尽层宽度

**突变结**：
$$W = \sqrt{\frac{2\epsilon_s}{q} \left(\frac{1}{N_A} + \frac{1}{N_D}\right) (V_{bi} - V)}$$

**单边突变结**（$N_A \gg N_D$，如p⁺n结）：
$$W \approx \sqrt{\frac{2\epsilon_s (V_{bi} - V)}{qN_D}}$$

耗尽层主要扩展在轻掺杂一侧。

**数值示例**：
- p⁺n结，$N_D = 10^{16}$ cm⁻³
- 零偏压，$V_{bi} = 0.7$ V
- $\epsilon_s = 11.9 \times 8.85 \times 10^{-14}$ F/cm

$$W = \sqrt{\frac{2 \times 1.05 \times 10^{-12} \times 0.7}{1.6 \times 10^{-19} \times 10^{16}}} \approx 0.3 \text{ μm}$$

### 击穿电压设计

**击穿机制**：

1. **雪崩击穿**（Avalanche Breakdown）
   - 高电场下载流子获得足够动能
   - 碰撞电离产生电子-空穴对
   - 主要发生在轻掺杂侧

2. **齐纳击穿**（Zener Breakdown）
   - 隧道效应
   - 主要发生在重掺杂结（< 5V）

**击穿电压与掺杂的关系**：
$$V_B \propto \frac{1}{N^{0.7 \sim 0.8}}$$

| 掺杂浓度 (cm⁻³) | 击穿电压 (V) |
|----------------|-------------|
| $10^{14}$ | 1000+ |
| $10^{15}$ | 300 |
| $10^{16}$ | 50 |
| $10^{17}$ | 10 |

## 双极型晶体管中的掺杂

### BJT结构

```
        发射极(E)    基极(B)     集电极(C)
           ████████████
        n⁺ │   p    │   n
           │  (薄)  │
           └────────┘

        发射结(JE)    集电结(JC)
```

### 掺杂设计原则

| 区域 | 掺杂类型 | 浓度范围 | 作用 |
|------|---------|---------|------|
| **发射区** | n⁺型 | $10^{19} \sim 10^{20}$ cm⁻³ | 高注入效率 |
| **基区** | p型 | $10^{16} \sim 10^{18}$ cm⁻³ | 薄基区，高增益 |
| **集电区** | n型 | $10^{15} \sim 10^{16}$ cm⁻³ | 高击穿电压 |

### 电流增益与掺杂

**共发射极电流增益**：
$$\beta = \frac{I_C}{I_B} \approx \frac{D_n}{D_p} \cdot \frac{N_E}{N_B} \cdot \frac{W_E}{W_B}$$

为提高增益：
- 发射区重掺杂（$N_E \gg N_B$）
- 基区薄且轻掺杂
- 使用多晶硅发射极

## MOSFET中的掺杂工程

### CMOS结构

```
        PMOS                  NMOS

   p⁺    p⁺              n⁺    n⁺
   源    漏               源    漏
    │─────│                │─────│
    │  n  │                │  p  │
    │ 阱  │                │ 阱  │
    └─────┘                └─────┘
      衬底p⁺                衬底n⁺
```

### 关键掺杂区域

**1. 阱（Well）**
- n阱：用于PMOS的p型衬底
- p阱：用于NMOS的n型衬底
- 深度：1-3 μm
- 浓度：$10^{16} \sim 10^{17}$ cm⁻³

**2. 源/漏区**
- 重掺杂（$10^{20}$ cm⁻³）
- 浅结（< 100 nm，先进工艺< 20 nm）
- 采用LDD（轻掺杂漏）结构

**3. 沟道区**
- 阈值电压调节注入
- 抗穿通注入
- 浓度：$10^{17} \sim 10^{18}$ cm⁻³

**4. 晕环/口袋注入（Halo/Pocket）**
- 沟道边缘的额外掺杂
- 抑制短沟道效应
- 非均匀掺杂分布

### LDD结构

**轻掺杂漏（Lightly Doped Drain）**：

```
传统结构：                    LDD结构：

    n⁺        n⁺                 n⁺   n⁻   n⁺
    ────────────                 ────┬────
                                    │
                                    │
                                    └─ 轻掺杂扩展区
```

**优点**：
- 降低漏区电场峰值
- 抑制热载流子效应
- 提高器件可靠性

## 先进掺杂技术

### 超浅结技术

**需求**：
- 短沟道器件需要超浅结（< 20 nm）
- 抑制短沟道效应

**技术方案**：
1. **低能离子注入**：1 keV以下B⁺注入
2. **等离子体掺杂**：无加速，极浅分布
3. **激光退火**：毫秒级退火，无扩散

### 应变工程与掺杂

**原理**：
- 在沟道引入应变
- 改变载流子有效质量
- 提高迁移率

**实现方式**：
- 源漏区外延SiGe（PMOS压应变）
- 源漏区外延SiC（NMOS张应变）
- 外延层同时完成掺杂

### 三维结构掺杂

**FinFET中的掺杂**：

```
     ┌───┐
     │   │ ← Fin（鳍）
     │   │
   ──┴───┴──

环绕栅极，沟道在Fin的三个表面
```

**掺杂挑战**：
- 三维结构的均匀掺杂
- Fin的侧壁与顶部掺杂差异
- 采用倾斜注入或扩散源

|------|---------|
