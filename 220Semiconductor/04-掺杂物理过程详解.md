# 04 - 掺杂物理过程详解

> 本章详细介绍半导体工艺中的各种掺杂技术，包括扩散、离子注入、外延生长等方法的物理原理和工艺特点。

---

## 掺杂的目的与意义

### 为什么需要掺杂？

**本征半导体的局限**：
- 载流子浓度低（Si: $n_i \approx 10^{10}$ cm⁻³）
- 电导率难以精确控制
- 无法形成pn结等器件结构

**掺杂带来的改变**：
- 载流子浓度提高 $10^6 \sim 10^{10}$ 倍
- 电导率可调范围超过10个数量级
- 可形成n型和p型区域，构建各种器件

### 掺杂的主要目的

| 目的 | 具体应用 |
|------|---------|
| **控制电导率** | 电阻、互连线、欧姆接触 |
| **形成pn结** | 二极管、晶体管、太阳能电池 |
| **阈值电压调节** | MOSFET沟道掺杂 |
| **防止穿通** | 阱区掺杂、隔离 |
| **降低接触电阻** | 重掺杂接触区 |

## 扩散掺杂（Diffusion）

### 扩散的基本原理

#### 菲克定律（Fick's Laws）

**菲克第一定律**（稳态扩散）：
$$J = -D \frac{\partial C}{\partial x}$$

其中：
- $J$：扩散通量（atoms/cm²·s）
- $D$：扩散系数（cm²/s）
- $C$：杂质浓度（atoms/cm³）
- 负号表示从高浓度向低浓度扩散

**菲克第二定律**（非稳态扩散）：
$$\frac{\partial C}{\partial t} = D \frac{\partial^2 C}{\partial x^2}$$

#### 扩散系数

扩散系数强烈依赖于温度：
$$D = D_0 \exp\left(-\frac{E_a}{k_BT}\right)$$

其中：
- $D_0$：前置因子（pre-exponential factor）
- $E_a$：激活能（activation energy）
- $T$：绝对温度

**硅中常见杂质的扩散参数**：

| 杂质 | $D_0$ (cm²/s) | $E_a$ (eV) | 1000°C时的D (cm²/s) |
|------|---------------|-----------|---------------------|
| 硼（B） | 0.76 | 3.46 | $1.5 \times 10^{-14}$ |
| 磷（P） | 3.85 | 3.66 | $2.3 \times 10^{-14}$ |
| 砷（As） | 24 | 4.08 | $1.2 \times 10^{-15}$ |
| 锑（Sb） | 0.214 | 3.65 | $1.3 \times 10^{-15}$ |

**温度依赖性**：温度每升高100°C，扩散系数约增加一个数量级。

### 扩散分布 profile

#### 1. 恒定表面浓度扩散

**边界条件**：
- $C(0,t) = C_s$（表面浓度恒定）
- $C(\infty,t) = 0$

**解**（余误差函数分布）：
$$C(x,t) = C_s \cdot \text{erfc}\left(\frac{x}{2\sqrt{Dt}}\right)$$

其中 $\text{erfc}(z) = 1 - \text{erf}(z)$ 为余误差函数。

**特征长度**：
- 扩散长度：$L = 2\sqrt{Dt}$
- 结深：$x_j = 2\sqrt{Dt} \cdot \text{erfc}^{-1}\left(\frac{C_{sub}}{C_s}\right)$

```
浓度
 C_s|██
    |███
    |████
    |██████
    |████████
 C_b|████████████
    |────────────────→ x
    0           x_j

余误差函数分布示意
```

#### 2. 恒定掺杂总量扩散（Delta函数源）

**初始条件**：
- $t=0$时，杂质以delta函数形式存在于表面：$C(x,0) = Q \delta(x)$
- $Q$：单位面积掺杂总量（dose）

**解**（高斯分布）：
$$C(x,t) = \frac{Q}{\sqrt{\pi Dt}} \exp\left(-\frac{x^2}{4Dt}\right)$$

**特征**：
- 峰值浓度：$C_p = \frac{Q}{\sqrt{\pi Dt}}$
- 峰值位置：$x=0$（表面）
- 标准差：$\sigma = \sqrt{2Dt}$

```

    |  ███████
    | █████████  C_p
    |███████████
    | █████████
    |  ███████
    |    ███
    |────────────────→ x
    0

高斯分布示意
```

### 扩散工艺参数

#### 温度

**典型扩散温度范围**：900°C ~ 1200°C

**温度选择考虑**：
- 温度越高，扩散系数越大
- 温度越高，固溶度越大
- 温度越高，晶格损伤恢复越好
- 但温度越高，热预算越大，横向扩散越严重

#### 时间

**扩散时间与结深的关系**：
$$x_j \propto \sqrt{t}$$

要获得2倍结深，需要4倍扩散时间。

#### 扩散源

| 杂质 | 扩散源 | 状态 |
|------|--------|------|
| 硼 | BBr₃, BCl₃ | 液态源 |
| 硼 | BN | 固态源 |
| 磷 | POCl₃ | 液态源 |
| 磷 | PH₃ | 气态源 |
| 砷 | AsH₃ | 气态源 |

### 两步扩散工艺

实际工艺通常采用两步法：

**第一步：预沉积（Predeposition）**
- 低温、短时间
- 高表面浓度
- 控制掺杂总量

**第二步：推进/驱入（Drive-in）**
- 高温、长时间
- 杂质再分布
- 达到所需结深和浓度分布

**数学描述**：
- 预沉积后：$C(x,0) = C_0 \cdot \text{erfc}(x/L_1)$
- 驱入后：高斯分布

## 离子注入（Ion Implantation）

### 离子注入原理

#### 基本过程

1. **离子源产生**：杂质原子电离形成离子（如P⁺, B⁺）
2. **加速**：离子在电场中加速到高能（keV ~ MeV）
3. **质量分析**：用磁场筛选特定质量的离子
4. **扫描注入**：离子束扫描整个晶圆表面
5. **进入晶格**：高能离子穿透进入硅晶体

#### 优势（与扩散相比）

| 特性 | 扩散 | 离子注入 |
|------|------|---------|
| 温度 | 900-1200°C | 室温（退火需高温） |
| 掺杂控制 | 受温度影响大 | 精确控制（剂量、能量） |
| 掺杂分布 | 受热力学限制 | 可超固溶度掺杂 |
| 横向扩散 | 严重 | 极小 |
| 浅结 | 难以实现 | 容易实现 |
| 损伤 | 小 | 大（需退火） |

### 射程与分布

#### 阻止机制

离子在固体中损失能量主要通过：

**1. 核阻止（Nuclear Stopping）**
- 离子与靶原子核的弹性碰撞
- 能量转移给晶格原子
- 导致晶格损伤
- 低能时占主导

**2. 电子阻止（Electronic Stopping）**
- 离子与靶电子的非弹性碰撞
- 激发电子或等离子体
- 高能时占主导

**总阻止本领**：
$$S(E) = S_n(E) + S_e(E) = -\frac{dE}{dx}$$

#### LSS理论

Lindhard, Scharff, Schiøtt (LSS) 理论描述了离子注入的统计分布。

**投影射程**（Projected Range）：
- 离子在入射方向的平均穿透深度：$R_p$
- 依赖于离子质量、能量和靶材料

**投影射程离散**（Straggle）：
- 射程的标准差：$\Delta R_p$
- 表示分布的展宽

**横向离散**（Lateral Straggle）：
- 垂直于入射方向的离散：$\Delta R_\perp$
- 决定掩膜边缘的过渡区宽度

#### 高斯分布近似

在低剂量、非晶靶的情况下，杂质浓度分布近似为高斯：

$$N(x) = \frac{Q}{\sqrt{2\pi}\Delta R_p} \exp\left[-\frac{(x-R_p)^2}{2\Delta R_p^2}\right]$$

其中 $Q$ 为注入剂量（ions/cm²）。

**硅中常用注入参数**：

| 离子 | 能量 (keV) | $R_p$ (nm) | $\Delta R_p$ (nm) |
|------|-----------|-----------|------------------|
| B⁺ | 10 | 33 | 13 |
| B⁺ | 50 | 160 | 45 |
| B⁺ | 100 | 320 | 75 |
| P⁺ | 50 | 60 | 20 |
| P⁺ | 100 | 120 | 40 |
| As⁺ | 50 | 33 | 11 |
| As⁺ | 100 | 66 | 22 |

### 沟道效应与损伤

#### 沟道效应（Channeling）

**现象**：
当离子沿晶体的某个低指数晶向（如[110]）入射时，
离子可能"穿行"于晶格原子之间的"沟道"中，
受到较小的阻止作用，射程显著增加。

```
无沟道效应：            有沟道效应：
    ● ○ ● ○ ●              ●   ●   ●
     \|/                   │ ○ │
    ─●─●─●─                │   │
      ○                    ○   ○
    射程正常               射程异常增加
```

**抑制方法**：
1. **倾斜注入**：晶圆倾斜7°左右偏离沟道方向
2. **氧化层覆盖**：表面SiO₂层使离子先发生随机碰撞
3. **预非晶化**：先注入Si或Ge破坏晶格周期性

#### 晶格损伤

**损伤的形成**：
- 每个注入离子造成数百到数千个晶格位移
- 形成空位-间隙原子对（Frenkel缺陷）
- 高剂量时形成非晶层

**损伤类型**：
| 剂量范围 | 损伤特征 |
|---------|---------|
| $<10^{12}$ cm⁻² | 孤立的点缺陷 |
| $10^{12} \sim 10^{14}$ cm⁻² | 缺陷团簇 |
| $>10^{15}$ cm⁻² | 非晶层形成 |

### 退火处理（Annealing）

退火的目的：
1. **激活杂质**：使杂质原子进入替位位置，成为电活性中心
2. **恢复晶格**：修复注入造成的晶格损伤
3. **消除缺陷**：减少载流子复合中心

#### 传统热退火

**温度**：600°C ~ 1100°C
**时间**：几分钟到几十分钟

**问题**：
- 杂质再分布（扩散）
- 热预算大
- 浅结变深

#### 快速热退火（RTA）

**特点**：
- 高温（~1000°C）
- 短时间（秒级）
- 快速升降温（~100°C/s）

**优点**：
- 杂质激活率高
- 扩散小，保持浅结
- 热预算低

**实现方式**：
- 卤钨灯加热
- 激光退火
- 电子束退火

## 外延生长掺杂

### 外延生长简介

**外延（Epitaxy）**：在单晶衬底上生长一层与衬底晶格取向一致的单晶薄膜。

**外延层的优势**：
- 可控的掺杂浓度（与衬底不同）
- 可控的厚度（nm到μm量级）
- 高质量的晶体结构
- 可生长异质结构

### 气相外延（VPE）

**硅气相外延常用反应**：
$$\text{SiCl}_4 + 2\text{H}_2 \rightarrow \text{Si} + 4\text{HCl} \quad (1200°C)$$
$$\text{SiH}_4 \rightarrow \text{Si} + 2\text{H}_2 \quad (600-800°C)$$

**原位掺杂**：
- n型：加入PH₃或AsH₃
- p型：加入B₂H₆

**掺杂控制**：
- 通过调节掺杂源气体流量
- 掺杂浓度可控制在 $10^{14} \sim 10^{20}$ cm⁻³
- 外延层与衬底界面突变

### 分子束外延（MBE）

**原理**：在超高真空（$<10^{-8}$ Pa）下，
将组成元素以原子束或分子束形式射向加热的衬底表面，
生长单晶薄膜。

**特点**：
- 生长温度低（400-800°C）
- 生长速率慢（μm/小时）
- 可精确控制层厚（单原子层）
- 适合III-V族化合物半导体

## 先进掺杂技术

### 等离子体浸没离子注入（PIII）

**原理**：
- 晶圆置于等离子体中
- 加负脉冲电压，等离子体中的离子被加速注入
- 全方向注入，适合三维结构

**优点**：
- 无质量分析器，成本低
- 适合大面积注入
- 可注入复杂形状结构

### 气体簇离子束（GCIB）

**原理**：
- 数千个气体原子形成簇
- 整体加速到高能
- 撞击表面时分散，减少损伤

**应用**：
- 超浅结形成
- 表面平滑
- 低损伤掺杂

### 单原子层掺杂（Delta Doping）

**原理**：
在生长过程中，中断主体材料的供给，
通入掺杂剂，在单原子层内引入高浓度掺杂，
然后恢复主体材料生长。

**应用**：
- 高电子迁移率晶体管（HEMT）
- 二维电子气（2DEG）
