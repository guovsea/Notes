# 03 - p型半导体与受主掺杂

> 本章深入讲解p型半导体的物理机制，重点阐述受主杂质（Acceptor）的作用原理和空穴导电的本质。

---

## p型半导体的定义

### 定义与命名

**p型半导体（p-type Semiconductor）**：
- 通过掺入**受主杂质**（Acceptor Impurities）而获得的半导体
- **多数载流子**（Majority Carriers）为**空穴**
- "p"代表"positive"，指带正电的载流子

### 核心特征

| 特征 | 描述 | 物理意义 |
|------|------|---------|
| **多数载流子** | 空穴（p >> n） | 空穴浓度远高于电子浓度 |
| **费米能级位置** | 靠近价带顶 | 表明电子占据价带概率高（空穴多） |
| **导电机制** | 空穴导电 | 价带中电子的移动形成空穴流动 |
| **掺杂元素** | 第III族元素 | 硼(B)、铝(Al)、镓(Ga)、铟(In) |

### p型与n型的对比

| 特性 | n型半导体 | p型半导体 |
|------|----------|----------|
| 掺杂元素 | 第V族（P, As, Sb） | 第III族（B, Al, Ga, In） |
| 多数载流子 | 电子 | 空穴 |
| 少数载流子 | 空穴 | 电子 |
| 费米能级位置 | 靠近导带底 | 靠近价带顶 |
| 电离后杂质电荷 | 正（$N_D^+$） | 负（$N_A^-$） |

## 受主杂质（Acceptor）

### 受主杂质的定义

**受主杂质（Acceptor）**：能够接受电子、增加价带中空穴浓度的杂质原子。

**命名由来**：
- "Acceptor"意为"接受者"
- 杂质原子从价带"接受"一个电子
- 自身成为带负电的离子
- 价带中留下空穴

### 第III族受主杂质

在硅中常用的受主杂质：

| 元素 | 原子序数 | 原子半径（nm） | 电离能（meV） | 特点 |
|------|---------|---------------|--------------|------|
| **硼（B）** | 5 | 0.088 | 45 | 最常用，扩散系数适中，固溶度高 |
| **铝（Al）** | 13 | 0.126 | 57 | 原子半径较大，易引入应变 |
| **镓（Ga）** | 31 | 0.126 | 65 | 常用于GaAs等化合物半导体 |
| **铟（In）** | 49 | 0.144 | 160 | 电离能较大，室温未完全电离 |

### 受主掺杂的原子物理图像

#### 1. 替位式掺杂

受主原子取代晶格中的硅原子位置：

```
        Si              Si              Si
        |               |               |
   Si—Si—Si    →   Si—B⁻—Si    +   h⁺（空穴）
        |               |               |
        Si              ○               Si
                       （缺失电子）

     掺杂前          掺杂后（电离后）
```

#### 2. 成键与缺电子

- 受主原子有**3个价电子**
- 与周围4个Si原子形成共价键需要4个电子
- 缺少**1个电子**来完成共价键网络
- 这个"缺电子"状态就是**空穴**

#### 3. 空穴的束缚与电离

**未电离时**：
- 受主原子从邻近Si-Si键中"借用"一个电子
- 在受主原子周围形成**负电荷中心**
- 邻近Si原子处留下空穴
- 空穴被受主离子的库仑力**弱束缚**

**电离过程**：
- 价带电子获得能量，填充受主原子的空位
- 受主原子变为负离子（$B^-$）
- 价带中产生空穴
- 空穴可在晶体中自由移动

### 受主能级的形成

**类氢模型近似**：

受主杂质也可以类比于类氢系统：
- 受主离子（$B^-$）相当于负电荷中心
- 空穴相当于正电荷粒子
- 空穴的有效质量为 $m_p^*$（Si: 0.49$m_0$）

**受主电离能计算**：

$$E_A = \frac{m_p^*}{m_0} \cdot \frac{1}{\epsilon_r^2} \cdot 13.6 \text{ eV}$$

对于硅中的硼：
$$E_A \approx \frac{0.49}{11.9^2} \times 13.6 \text{ eV} \approx 47 \text{ meV}$$

（实验值约45 meV）

### 受主能级在能带图中的位置

```
能量 ↑
     |    ● E_A （受主能级）
     |    |
     |    |
 E_A |    |     ┌──────────── 导带底 E_c
     |    |     |
     |    |     |
     |    ●─────┘──────────── 价带顶 E_v
     |
     +──────────────────→ 位置
```

**关键数据（硅中）**：
| 受主杂质 | 电离能（meV） | 能级位置（价带上） |
|---------|--------------|-------------------|
| 硼（B） | 45 | $E_v + 0.045$ eV |
| 铝（Al）| 57 | $E_v + 0.057$ eV |
| 镓（Ga）| 65 | $E_v + 0.065$ eV |
| 铟（In）| 160 | $E_v + 0.16$ eV |

**结论**：受主能级非常靠近价带顶，室温下硼、铝、镓几乎完全电离，铟由于电离能较大，室温下电离率约90%。

## 多数载流子与少数载流子

### 载流子浓度的基本关系

在p型半导体中：

**1. 多数载流子（Majority Carriers）**
- **类型**：空穴
- **浓度**：$p_p$（下标p表示p型）
- **来源**：受主杂质电离 + 本征激发

**2. 少数载流子（Minority Carriers）**
- **类型**：电子
- **浓度**：$n_p$
- **来源**：仅由本征激发产生

### 电中性条件

对于只含受主杂质的p型半导体：
$$p_p + N_D^+ = n_p + N_A^-$$

简化为（$N_D = 0$）：
$$p_p = n_p + N_A^-$$

### 载流子浓度计算

#### 完全电离近似

室温下，受主杂质（B, Al, Ga）几乎全部电离：
$$N_A^- \approx N_A$$

电中性条件：
$$p_p \approx N_A + n_p$$

由于 $p_p \gg n_p$：
$$p_p \approx N_A$$

#### 质量作用定律

$$p_p \cdot n_p = n_i^2$$

因此，少数载流子浓度：
$$n_p = \frac{n_i^2}{p_p} \approx \frac{n_i^2}{N_A}$$

### 数值示例

**例**：硅中掺入硼，浓度 $N_A = 10^{16}$ cm⁻³，T = 300K

**计算**：
- 空穴浓度：$p_p \approx N_A = 10^{16}$ cm⁻³
- 电子浓度：$n_p = \frac{(1.5 \times 10^{10})^2}{10^{16}} = 2.25 \times 10^{4}$ cm⁻³
- 浓度比：$\frac{p_p}{n_p} \approx 4.4 \times 10^{11}$

### 铟的特殊情况

由于铟的电离能较大（160 meV），室温下不完全电离：

**电离率公式**：
$$\frac{N_A^-}{N_A} = \frac{1}{1 + g_A \exp\left(\frac{E_A - E_F}{k_BT}\right)}$$

其中 $g_A = 4$（价带轻/重空穴的简并）。

室温下铟的电离率约90%，因此在精确计算中：
$$p_p \approx 0.9 N_A$$

这是实际工艺中主要使用硼而非铟的原因之一。

## 受主掺杂的物理过程

### 空穴的物理本质

**重要理解**：空穴不是一种"粒子"，而是一种**集体激发态**。

#### 价带电子运动的等效描述

当价带中存在空位（空穴）时：

**微观图像**：
- 周围价电子可以填充这个空位
- 空位在晶格中"移动"
- 共有 $N-1$ 个电子运动（$N$ 为价键数）

**等效描述**：
- 一个"空穴"在晶格中移动
- 空穴带正电荷 $+q$
- 空穴具有**正有效质量**

#### 空穴的有效质量

**为什么空穴有正有效质量？**

价带顶附近的E-k关系为**向下凹**：
$$E(k) = E_v - \frac{\hbar^2 k^2}{2m_p^*}$$

由 $F = \frac{\hbar^2}{m^*} \frac{dk}{dt}$ 和 $m^* = \frac{\hbar^2}{\partial^2 E / \partial k^2}$：
- 价带顶的曲率为负
- 有效质量 $m_p^* > 0$（取正值）
- 空穴在外电场下沿电场方向移动

### 受主电离的详细过程

#### 电离前后的能带图变化

**未电离时**：
```
    导带
    ─────

    受主能级 E_A
    ●
    |
    ○ 空穴（局域化）

    价带
    ─────
```

**电离后**：
```
    导带
    ─────



    价带
    ───── ○ 空穴（自由移动）
```

#### 电离能量分析

电离能 $E_A$ 定义为：
$$E_A = E_A^{level} - E_v$$

即把价带电子激发到受主能级所需的能量。

物理过程：
1. 价带电子获得能量 $E_A$
2. 电子跃迁填充受主能级
3. 价带产生空穴
4. 空穴成为自由载流子

## p型硅的制备与应用

### 典型掺杂浓度

| 应用 | 掺杂浓度（cm⁻³） | 用途 |
|------|-----------------|------|
| 轻掺杂p型 | $10^{14} \sim 10^{16}$ | 衬底、电阻 |
| 中掺杂p型 | $10^{16} \sim 10^{18}$ | MOSFET沟道（PMOS） |
| 重掺杂p型 | $10^{19} \sim 10^{20}$ | 接触、基区、源漏区 |

### 硼掺杂工艺

**1. 硼扩散**
- 源：BBr₃、B₂H₆ 或 BN 固体源
- 温度：900-1100°C
- 特点：扩散速度快，适合深结

**2. 硼离子注入**
- 注入离子：B⁺
- 能量：keV 到 MeV 量级
- 特点：剂量精确控制，浅结

**3. 外延掺杂**
- 源：B₂H₆（乙硼烷）
- 特点：原位掺杂，浓度均匀

### CMOS工艺中的p型区

在CMOS集成电路中，p型掺杂用于：
1. **PMOS晶体管**：p⁺源/漏区
2. **n阱**：PMOS所在的p型衬底或n阱中的p型区
3. **p型基区**：双极型晶体管
4. **电阻**：多晶硅电阻或扩散电阻

## p型与n型的对称性

### 电子-空穴对称性

n型和p型半导体具有深刻的对称性：

| 方面 | n型 | p型 |
|------|-----|-----|
| 载流子 | 电子 | 空穴 |
| 电荷 | -q | +q |
| 有效质量 | $m_n^*$ | $m_p^*$ |
| 态密度 | $N_c$ | $N_v$ |
| 杂质 | 施主 | 受主 |
| 能级 | 导带底下 | 价带顶上 |
| 费米能级移动 | 向导带 | 向价带 |

### 统一描述

采用**费米-狄拉克分布**可以统一描述两种类型：

电子浓度：
$$n = N_c F_{1/2}\left(\frac{E_F - E_c}{k_BT}\right)$$

空穴浓度：
$$p = N_v F_{1/2}\left(\frac{E_v - E_F}{k_BT}\right)$$

当 $E_F$ 靠近 $E_c$ 时，$n > p$，为n型；
当 $E_F$ 靠近 $E_v$ 时，$p > n$，为p型。

## 本章小结

| 概念 | 核心要点 |
|------|---------|
| **p型半导体** | 空穴为多数载流子，由受主掺杂获得 |
| **受主杂质** | 第III族元素（B, Al, Ga, In），接受电子产生空穴 |
| **受主能级** | 位于价带顶上方~45-160 meV |
| **载流子浓度** | $p_p \approx N_A$，$n_p = n_i^2/N_A$ |
| **空穴本质** | 价带集体运动的等效描述，带正电、正有效质量 |

**关键公式**：
- 电中性：$p_p = N_A^- + n_p$
- 质量作用定律：$p_p \cdot n_p = n_i^2$
- 受主电离能：$E_A \approx \frac{m_p^*}{m_0 \epsilon_r^2} \times 13.6$ eV

**重要区别**：
- 硼：电离能45 meV，室温完全电离，最常用
- 铟：电离能160 meV，室温不完全电离，少用
