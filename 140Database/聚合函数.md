SQL 聚合函数计算一组值并**返回单个值**。因为聚合函数对一组值进行操作，所以它通常与`SELECT`语句的 `GROUP BY` 子句一起使用。 `GROUP BY`子句将结果集划分为值分组，聚合函数为每个分组返回单个值。不分组则对整个查询结果进行求值。

- COUNT 
- MAX
- MIN
- AVG
- SUM

`COUNT()`函数外，SQL 聚合函数忽略`null`值。聚合函数只能作用于集合对象，不能作用于单行。SELECT 语句和 HAVING 语句的作用对象是选择之后或者分组之后的集合，而 WHERE 语句的作用对象是单行。因此，聚合函数用作表达式**不能作用在 WHERE** 。只能作用在 **SELECT** 和 **HAVING**。


**聚合函数的一般用法**

聚合函数([ALL | DISTINCT] 列名 ｜ 表达式) 

默认修饰词指定是否计算重复值，默认为 ALL 计算重复值。

**COUNT**
```sql
    COUNT ( [ALL | DISTINCT] 列名 | 表达式 | *)
```

**AVG**
```sql
AVG( [ALL | DISTINCT] 列名 | 表达式)
```
`ALL`关键字指示`AVG()`函数计算所有值的平均值，而`DISTINCT`关键字强制函数仅对不同的值进行操作。 默认情况下，使用`ALL`选项

**SUM**

```sql
SUM([ALL | DISTINCT] 列名 ｜ 表达式)
```
**MIN/MAX**

MIN 也可以用 **DISTINCT** 修饰，但是没有意义
```sql
MIN([ALL | DISTINCT] 列名 | 表达式)
```
