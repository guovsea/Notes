存储函数是有返回值的存储过程，存储函数的参数只能是IN类型的。具体语法如下:

**语法**
```sql
CREATE FUNCTION 存储函数名称 ([ 参数列表 ]) 
RETURNS type [characteristic ...]  
BEGIN 
    SQL语句
    RETURN 返回值; 
END;
```

characteristic说明:
- DETERMINISTIC：相同的输入参数总是产生相同的结果 (deterministic)，说明该存储函数是确定的，可以用 [[binlog#4.binlog 格式|SEGMENT 模式]]保存而不需要上下文
- NO SQL：不包含 SQL 语句。  
- READS SQL DATA：包含读取数据的语句，但不包含写入数据的语句。

**案例**
```sql
create function fun1(n int)
returns int deterministic
begin

    declare total int default 0;

while n>0 do  
set total := total + n; set n := n - 1;

end while;

    return total;
end;

select fun1(50);
```
在 mysql8.0版本中 log bin 默认是开启的，log bin 一旦开启了，mysql 就要求在定义存储过程时，需要指定 characteristic 特性，否则就会报错。
