```sql
CAST(值 AS 类型)   -- 经过测试，如果想转成整数的话，类型只能是SIGNED或者UNSIGNED
CONVERT(值, 类型)  -- 字符串到SIGNED和UNSIGNED很智能，反正就是”尽量转”
                  -- '123'->123, '123b'->123, '-123'-> -123 'f123'->0
```

## 可转换的类型

| 类型 | 说明 |
|---|---|
| BINARY | 转为二进制字符串 |
| CHAR | 转为字符串 |
| DATE | 转为日期 |
| DATETIME | 转为日期时间 |
| TIME | 转为时间 |
| DECIMAL | 转为浮点数 |
| SIGNED | 转为有符号整数 |
| UNSIGNED | 转为无符号整数 |
