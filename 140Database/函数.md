## 字符串函数

- `CONCAT (S1, S2,... Sn)` -- 拼接
- `LOWER(str)` -- 全部转小写
- `UPPER(str)` -- 全部转大写
- `LPAD(str,n,pad)` -- 左边填充 pad 字符使得**整个字符串的长度达到 n** 
- `RPAD(str,n,pad)`
- `TRIM(str)` -- 去掉字符串头部和尾部的空格
- `SUBSTRING(str,start,len)`  -- 返回从字符串str从start位置起的len个长度的字符串
- `REPLACE(str, old, new)` -- 替换字符串
## 数值函数

- `CEIL(x)`      -- 向上去整 (ceil 天花板)
- `FLOOR(x)`     -- 向下去整
- `MOD(x, y)`    -- x % y
- `RAND()`       -- 返回 0～1 的随机数
- `ROUND(x, d)`  -- x 保留 d 位小数
## 日期函数

- `CURDATE()`  -- 当前日期 `'YYYY-MM-DD'`
- `CURTIME()`  -- 当前时间 `'HH:MM:SS'`
- `NOW()`     -- 当前日期和时间 `'YYYY-MM-DD HH:MM:SS'`
- `YEAR(date)`
- `MONTH(date)`
- `DAY(date)`
- `DATE_ADD(date, INTERVAL expr TYPE)` -- 时间 + 间隔
```sql
SELECT date_add(cast(now() as date), INTERVAL 1 year);

SELECT date_add(cast(now() as time), INTERVAL 1 day);

SELECT date_add(cast(now() as datetime), INTERVAL 1 month);

SELECT date_add('2023-9-24', INTERVAL 1 week);

SELECT date_add(cast('00:56:48' as time), INTERVAL 1 hour);
-- date 可以是各种时间类型
-- expr 可以为负数/正数，整数小数
-- TYPE 可以是各种时间单位，second, minute, hour, day, week, month, year...
```
- `DATEDIFF(date1, date2)`  -- `date1 - date2`, 会返回`+/-`天数
- `TIMEDIFF(time1, time2)`  -- `time1 - time2`, 会返回`+/-'HH:MM:SS'`的差值
  返回的差值存在一个范围`-838:59:59～-838:59:59`（这是`time`类型的范围）

## 流程函数

可以在SQL语句中实现条件筛选

- `IF(value , t , f)`
- `IFNULL(value1 , value2)`
- `CASE WHEN`
-- 简单 `CASE` 函数**仅检查是否相等**类似于 c 语言中的`switch`

-- **`CASE` 搜索函数才会判断每个条件**
```sql
-- 简单 CASE 函数 简单 CASE 函数仅检查是否相等!!!
CASE 条件表达式 WHEN 值1 THEN 结果1 WHEN 值2 THEN 结果2 ... [ELSE 默认] END; 

CASE 条件表达式 
WHEN 值1 THEN 结果1 
WHEN 值2 THEN 结果2 
... 
[ELSE 默认] 
END; 

-- CASE 搜索函数
CASE WHEN 条件1 THEN 值1 WHEN 条件2 THEN 结果2 ... [ELSE 默认] END;

CASE
WHEN 条件1 THEN 结果1
WHEN 条件2 THEN 结果2
... 
[ELSE 默认] 
END;
```

## 示例代码

```sql
SELECT job_title,
    CASE 
    WHEN min_salary< 3000 THEN '低收入'
    WHEN min_salary< 5000 THEN '中收入'
    ELSE '高收入'
    END
FROM jobs;
```
