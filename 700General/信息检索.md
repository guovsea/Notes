## 提高搜索效率

**使用 Google 引擎**

**搜索语法**

| 语法          | 含义｜                         |
| ------------- | ------------------------------ |
| 英文""        | 精确匹配引号中的内容           |
| +/AND         | 和                             |
| OR            | 或者                           |
| -             | 不包括后面的内容               |
| *             | 通配符                         |
| @             | @社交媒体，限定社交媒体        |
| #             | \#标签 ，搜索某一标签          |
| ..            | 限定范围，*数字 1.. 数字 2*    |
| `intitle:`    | 针对标题进行搜索               |
| `allintitle:` | 使用多个限定词针对标题进行搜索 |
| `inurl: `     | 针对 url 进行搜索              |
| `site:`       | 限定某个网站                   |
| `imagesize:`  | 限定图片大小，*2560x1440*      |
| `filetype:`   | 指定文件类型                   |
