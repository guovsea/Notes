2. 普通函数不能使用静态成员变量
```cpp
int SR::Router::AddRouting(SR::IPAddr destination, SR::IPAddr mask, std::string portName, SR::IPAddr nextHop) {
    assert(!HasRouting(destination, mask, portName, nextHop));
    assert(HasPort(portName));
    m_sCurrentRTEntryIndex++;
    m_routingTable.emplace_back(0,destination, mask, GetPort(portName), nextHop);
    return m_routingTable.size() - 1;
}
```

如果声明有静态成员变量，一定要在**源文件**中定义（而不是在头文件中），否则会报错
```cpp

# .h 
struct Temp
{
    void func(void) {
        cnt++;
    }
    static int cnt;  // 声明静态成员变量
};
# .h
int Temp::cnt = 0;  // 定义静态成员变量


:-1: error: symbol(s) not found for architecture x86_64

```
