**函数返回时会在调用点构造返回值类型的对象**。

例如，下面这个函数：

```cpp
T func() {
    return expr;
}
```

在返回时会根据 `expr` 调用*某种*构造函数构造一个对象，等价于调用 `T = expr`。
## 函数返回值用于构造对象

```cpp
T t = func();
```

可能发生的情况：

- 触发 RVO 机制
- 拷贝构造（构造-拷贝）
- 移动构造（构造-移动）

## 函数返回值用于构造对象

```cpp
t = func();
```

可能发生的情况：

- 拷贝构造（构造-拷贝赋值运算符）
- 移动构造（构造-移动赋值运算符）
