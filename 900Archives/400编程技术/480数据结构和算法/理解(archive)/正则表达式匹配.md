## 题目描述

给你一个字符串 s 和一个字符规律 p，请你来实现一个支持 '.' 和 '\*' 的正则表达式匹配。

- '.' 匹配任意单个字符
- '\*' 匹配零个或多个前面的那一个元素

所谓匹配，是要涵盖 整个 字符串 s的，而不是部分字符串。

算法思路：

考虑这样一个函数，`bool process(int i, int j)`。假设它能判断长度为i的s字符串和长度为j的p字符串是否匹配。

那么base case为：
- i=0（s字符串长度为0）时，则对于p字符串，如果最后一个字符为*，那么就能编程求解子问题process(0, j - 2)

```cpp
for (int j = 2; j <= pL; j++) {
    // 长度为j的p如果最后一个字符为*
    if (p[j - 1] == '*') {
        dp[0][j] = dp[0][j - 2];
    }
}
```
- i=1 &&j=1，s和p都只有一个字符时，这两个字符必须相等

递推条件为：

1. s末尾字符和p末尾字符可以直接匹配。可以直接匹配有两种情况，要么他们真的相等`s[i - 1] == p[j - 1]`，要么就是p末尾字符为'.'。这样问题process(i, j)就能转换为求子问题process(i-1, j-1)

```cpp
if (s[i - 1] == p[j - 1] || p[j - 1] == '.') {
    dp[i][j] = dp[i - 1][j - 1];
}
```

2. p末尾的字符为*。
   
此时需要考虑倒数第二个`p[j-2]`的情况，**如果`p[j-2]`能和s的末尾字符串匹配**（也分为1.中的两种情况），那么就需要考虑对`p[j-2]`进行几次匹配。在这里有三中情况：

    - 匹配0次
      
    直接让`a*`被消失掉，s不变，p的长度会-2，那么就是process(i, j) = process(i, j-2)
    
    - 匹配1次
      
    让`a*`消失掉，并且匹配一次a ，process(i, j) = process(i-1, j-2)
    
    - 匹配2次以以上
    
    保持`a*`不变，匹配一次a，s的长度-1，p的长度不变。process(i, j) = process(i-1, j)。这样`a*`结构还能继续匹配0次或者一次，因此就能匹配2次及以上

```cpp
if (s[i - 1] == p[j - 2] || p[j - 2] == '.') {
    dp[i][j] =
        dp[i][j - 2] || dp[i - 1][j - 2] || dp[i - 1][j];
} else {
```

**如果`p[j-2]`不能和s的末尾字符串匹配**，那么此时只能匹配0次了

```cpp
else {
    dp[i][j] = dp[i][j - 2];
}
```

3. 其他所有情况，都为false，因此dp最开始初始化为false，但需要注意下标范围为**0～Len**，因此`dp[sL1][pL+1]`

完整代码

```cpp
bool isMatch(string s, string p) {
    int sL = s.length(), pL = p.length();
    // 默认为false
    vector<vector<bool>> dp(20, vector<bool>(20, false));
    dp[0][0] = true;
    // s的长度为0时，p的长度至少为2才能匹配
    // s = "" , p = "a*" / "a*b*"
    for (int j = 2; j <= pL; j++) {
        // 长度为j的p如果最后一个字符为*
        if (p[j - 1] == '*') {
            dp[0][j] = dp[0][j - 2];
        }
    }
    // 当s和p的长度都为1时，只有相同才为true
    dp[1][1] = s[0] == p[0] ? true : false;
    for (int i = 1; i <= sL; i++) {
        for (int j = 1; j <= pL; j++) {
            // 直接匹配
            if (s[i - 1] == p[j - 1] || p[j - 1] == '.') {
                dp[i][j] = dp[i - 1][j - 1];
            // 字符p[j-1]为*
            }
            if (p[j - 1] == '*') {
                // *前一个字符可以被匹配
                if (s[i - 1] == p[j - 2] || p[j - 2] == '.') {
                    // 匹配0次，1次，n次
                    dp[i][j] =
                        dp[i][j - 2] || dp[i - 1][j - 2] || dp[i - 1][j];
                } else {
                    // *前一个字符不能被匹配，只能匹配0次
                    dp[i][j] = dp[i][j - 2];
                }
            }
        }
    }
return dp[sL][pL];
}
```
