## 多表关系

1. 一对一
   实现：在任意一方加入外键，关联另外一方的主键，并且设置外键为唯一的(UNIQUE)
2. 一对多   
   实现：在多的一方建立外键，指向一的一方的主键
3. 多对多
   实现：建立第三张中间表，中间表至少包含两个外键，分别关联两方主键

## 连接查询

- 内连接: 返回两张表都符合条件的部分  
- 外连接: 
  - 左外连接: 查询左表所有数据，以及右表中符合条件的数据
    就会像这样：
    
    | *左表中符合条件的数据* | *右表中符合条件的数据* |
    | -------------------- | -------------------- |
    | *左表中的其他数据*     | *NULL*                 |
    
  - 右外连接: 查询右表所有数据，以及左表中符合条件的数据

    | *左表中符合条件的数据* | *右表中符合条件的数据* |
    | -------------------- | -------------------- |
    |  *NULL*     |*右表中的其他数据*                 |
  - 自连接:当前表与自身的连接查询，自连接必须使用表别名
### 内连接

隐式内连接  
```sql
SELECT 字段列表 FROM 表1 , 表2 WHERE 连接条件;
```
显式内连接  
```sql
SELECT 字段列表 FROM 表1 [ INNER ] JOIN 表2 ON 连接条件;
```
### 外连接

`LEFT JOIN ->`保留左边的表中的所有信息，`RIGHT JOIN`-> 保留右边表中的所有信息
左外连接
```sql
SELECT 字段列表 FROM 表1 LEFT [OUTER] JOIN 表2 ON 连接条件;
```
右外连接
```sql
SELECT 字段列表 FROM 表1 RIGHT [OUTER] JOIN 表2 ON 连接条件;
```
### 自连接

多次连接表本身，必须起别名，可以是内连接、外连接
```sql
SELECT 字段列表 FROM 表 A 别名 A [INNER | LEFT | RIGHT] JOIN 表 A 别名 B ON 连接条件
```
## 联合查询
```sql
SELECT 字段列表 FROM 表A ...   -- 查询1 
UNION [ ALL ]   -- 默认会去重(所有字段完全一样)，ALL则不会去重
SELECT 字段列表 FROM 表B ....; -- 查询2
```
**如果两个查询结果结构不一致则会发生错误**

## 子查询

SQL 语句中嵌套 SELECT 语句，称为嵌套查询，又称子查询。子查询外部的语句可以是 INSERT / UPDATE / DELETE / SELECT 的任何一个。

根据子查询结果不同，分为:  
1. 标量子查询(子查询结果为单个值)
2. 列子查询(子查询结果为一列)  
3. 行子查询(子查询结果为一行)  
4. 表子查询 (子查询结果为多行多列)

**注意**

- 给出的条字段列表一定要和查询结果的字段结构相一致
- WHERE 条件语句的语意是对每一行的某些字段列表是否满足这个条件。因此只有条件中只有一个元素才能用比较运算符，有多个元素必须用集合运算符。

### 标量子查询

子查询返回的结果是单个值(数字、字符串、日期等)，最简单的形式。 常用的操作符:`= <> > >= < <=`

### 列子查询

子查询返回的结果是**一列(可以是多行)**。常用的操作符:` IN 、NOT IN 、 ANY 、SOME 、 ALL`因为列子查询的返回结果可以看做是一个集合。

- ANY 、SOME， ANY 等价与 SOME ，子查询返回列表中，有任意一个满足即可
```sql
select * from emp where salary > any(select salary from emp where dept_id = 1);
    -- 查询比部门编号为1的任意一人的工资都高的员工信息
```
- ALL 子查询返回列表的所有值都必须满足
### 行子查询

子查询返回的结果是**一行**(可以是多列)。常用的操作符: = 、<> 、IN 、NOT IN

案例:  
查询与 "张无忌" 的薪资及直属领导相同的员工信息 

1. 查询 "张无忌" 的薪资及直属领导
```sql
select salary, managerid from emp where name = '张无忌';
```
2. 查询与 "张无忌" 的薪资及直属领导相同的员工信息 ;
```sql
select * from emp where (salary,managerid) = (select salary, managerid from emp where name = '张无忌');
            -- 给出的条字段列表一定要和查询结果的字段结构相一致
```
**注意**

- 给出的条字段列表一定要和查询结果的字段结构相一致
