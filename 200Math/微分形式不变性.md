# 微分形式不变性

一阶微分形式不变性是指：无论 $u$ 是自变量还是中间变量，$y = f(u)$ 的微分总是 $dy = f'(u)\,du$。这一性质使得在复合函数求微分时，无需区分变量角色，形式上保持统一。这对链式法则的理解和多元全微分的推广至关重要。
