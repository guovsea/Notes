# 有限元

有限元方法（Finite Element Method, FEM）是一种求解偏微分方程的数值方法。其核心思想是将连续求解域离散为有限个互不重叠的子区域（单元），在每个单元上用简单的插值函数近似未知场变量，最终将偏微分方程转化为代数方程组进行求解。FEM 在结构力学、热传导、流体力学和电磁场等工程领域有广泛应用。
