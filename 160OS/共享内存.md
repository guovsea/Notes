## 概念

**将多个进程的虚拟地址空间映射到同一物理地址空间**

共享内存实际上就是进程通过调用 shmget（Shared Memory GET 获取共享内存）来分配一个共享内存块，然后每个进程通过 shmat（Shared Memory Attach 绑定到共享内存块），将进程的逻辑虚拟地址空间指向共享内存块中。随后需要访问这个共享内存块的进程都必须将这个共享内存绑定到自己的地址空间中去。

**共享内存的特点：**  

1. 共享内存是进程间共享数据的一种最快的方法。
2. 必须提供自己的同步措施

**实现过程**

- 分配共享内存块，会创建新的内存页面
- 绑定到分配的共享内存块，将进程中某块虚拟内存绑定到分配的共享内存块
- 解除共享内存绑定
- 必须有一个（且只能是一个）进程负责释放这个被共享的内存页面

## 分配共享内存

```c
#include <sys/ipc.h>
#include <sys/shm.h>
int shmget(key_t key, size_t size,int shmflg);  // 创建或打开一块共享内存区。
```

**参数：**

- key：进程间通信键值，可以使用 `ftok()` [[# ftok函数]]。
- size：该共享存储段的长度 (字节)。
- shmflg：标识函数的行为及共享内存的权限，其取值如下：
  - IPC_CREAT：如果不存在就创建
  - IPC_EXCL： 如果已经存在则返回失败

**返回值：**

- 成功：共享内存标识符。
- 失败：-1。

## 绑定共享内存

```c
#include <sys/types.h>
#include <sys/shm.h>
void *shmat(int shmid, const void *shmaddr, int shmflg);  // shm attach
```

**参数：**

- shmid：共享内存标识符，shmget () 的返回值。
- shmaddr：共享内存映射地址 (若为 NULL 则由系统自动指定)，一般使用 **NULL**。
- shmflg：共享内存段的访问权限和映射条件 ( 通常为 0 )，具体取值如下：
  - 0：共享内存具有可读可写权限。
  - SHM_RDONLY：只读。
  - lSHM_RND：（shmaddr 非空时才有效）

**返回值：**

- 成功：共享内存段映射地址 ( 相当于这个指针就指向此共享内存 )
- 失败：-1 *需要强转成 void，`if (shm_ptr == (void *)-1)`*

## 解除共享内存映射

如果当释放这个内存块的进程是最后一个使用该内存块的进程，则这个内存块将被删除。对 exit 或任何 exec 族函数的调用都会自动使进程脱离共享内存块。

```c
#include <sys/types.h>
#include <sys/shm.h>
int shmdt(const void *shmaddr);   // shm detach
```
**参数：**

- shmaddr：共享内存映射地址。

**返回值：**

- 成功：0
- 失败：-1

## 共享内存的控制

```c
#include <sys/ipc.h>
#include <sys/shm.h>
int shmctl(int shmid, int cmd, struct shmid_ds *buf);
```
**参数：**

- shmid：共享内存标识符。
- cmd：函数功能的控制，其取值如下：
  - IPC_RMID：删除。(常用 )
  - IPC_SET：设置 shmid_ds 参数，相当于把共享内存原来的属性值替换为 buf 里的属性值。
  - IPC_STAT：保存 shmid_ds 参数，把共享内存原来的属性值备份到 buf 里。
  - SHM_LOCK：用于锁定共享内存段，以确保其一直保留在物理内存中，而不被操作系统交换到磁盘上。( 超级用户权限)。
  - SHM_UNLOCK：解锁共享内存段。
- buf：shmid_ds 数据类型的地址 (具体类型请点此链接 )，用来存放或修改共享内存的属性。

**返回值：**

- 成功：0
- 失败：-1

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <sys/ipc.h>
#include <sys/shm.h>
#include <sys/types.h>
#include <unistd.h>

#define BUFSZ 512

int main(int argc, char *argv[]) {
    int shmid;
    int ret;
    key_t key;
    char *shmadd;

    // 创建key值
    key = ftok("../", 2015);
    if (key == -1) {
        perror("ftok");
    }

    // 创建共享内存
    shmid = shmget(key, BUFSZ, IPC_CREAT | 0666);
    if (shmid < 0) {
        perror("shmget");
        exit(-1);
    }

    // 映射
    shmadd = shmat(shmid, NULL, 0);
    if (shmadd < 0) {
        perror("shmat");
        _exit(-1);
    }

    // 拷贝数据至共享内存区
    printf("copy data to shared-memory\n");
    bzero(shmadd, BUFSZ); // 共享内存清空
    strcpy(shmadd, "how are you, lh\n");

    return 0;
}
```

```c
// 读端
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <sys/ipc.h>
#include <sys/shm.h>
#include <sys/types.h>
#include <unistd.h>

#define BUFSZ 512

int main(int argc, char *argv[]) {
    int shmid;
    int ret;
    key_t key;
    char *shmadd;

    // 创建key值
    key = ftok("../", 2015);
    if (key == -1) {
        perror("ftok");
    }

    system("ipcs -m"); // 查看共享内存

    // 打开共享内存
    shmid = shmget(key, BUFSZ, IPC_CREAT | 0666);
    if (shmid < 0) {
        perror("shmget");
        exit(-1);
    }

    // 映射
    shmadd = shmat(shmid, NULL, 0);
    if (shmadd < 0) {
        perror("shmat");
        exit(-1);
    }

    // 读共享内存区数据
    printf("data = [%s]\n", shmadd);

    // 分离共享内存和当前进程
    ret = shmdt(shmadd);
    if (ret < 0) {
        perror("shmdt");
        exit(1);
    } else {
        printf("deleted shared-memory\n");
    }

    // 删除共享内存
    shmctl(shmid, IPC_RMID, NULL);

    system("ipcs -m"); // 查看共享内存

    return 0;
}
```
