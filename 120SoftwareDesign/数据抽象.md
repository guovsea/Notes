数据类型指的是一组**值**和一组对这些值的**操作**的集合

数据抽象编程风格（还有函数抽象编程风格）可以在更高抽象层次上编写程序，更加方便。

数据抽象类型（ADT）是一种能够对使用者隐藏数据表示的数据类型，主要特征是将数据和函数实现关联。

## ADT 的核心思想

- **封装**：隐藏数据的具体表示方式，只暴露操作接口
- **信息隐藏**：使用者不需要知道数据内部如何存储和实现
- **接口与实现分离**：对外提供稳定的接口，内部实现可以自由变化

## 示例

```cpp
// 用 ADT 的方式定义一个栈
class Stack {
public:
    virtual void push(int value) = 0;
    virtual int pop() = 0;
    virtual int top() const = 0;
    virtual bool isEmpty() const = 0;
    virtual ~Stack() = default;
};

// 使用者只需要知道 Stack 的接口，不需要关心底层是用数组还是链表实现
```
