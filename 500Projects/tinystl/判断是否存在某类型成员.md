## 错误写法

错误点：

- 错误假定了 `T::value` 的类型是 `bool`
- 模板中的非类型参数（即使是在偏特化模板中）使用了 `T::value` 的形式
  >  `T::value` 也可能是静态变量，因此编译器会抱怨非类型参数使用了变量

```cpp
template<typename T, bool>
struct has_type : std::false_type{};

template<typename T>
struct has_type<T, T::value> : std::true_type {};
```

## 正确写法

`template <typename T, typename = void>`， void 不影响重载，是最泛化的主模板

偏特化模板中，在第二个类型参数中使用 `enable_if` ，触发 SFINAE

```cpp

template <typename T, typename = void>
struct has_type : std::false_type {};

template <typename T>
struct has_type<T, std::enable_if<T::value, void>> : std::true_type {};

```
