# LLVM 学习路线

## 学习步骤

1. **实现一门简单的编程语言**
   通过官方教程（Kaleidoscope）学习如何使用 LLVM 构建一门语言。
   [LLVM Tutorial: Table of Contents](https://llvm.org/docs/tutorial/)

2. **阅读 LLVM 核心程序员的文档**
   了解 LLVM 的核心数据结构和 API。
   [LLVM Programmer’s Manual](https://llvm.org/docs/ProgrammersManual.html)

3. **写一个自己的 LLVM Pass**
   学习如何编写 Pass 来分析和变换 IR。
