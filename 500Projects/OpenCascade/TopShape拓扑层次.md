# OpenCascade / FreeCAD 中 TopoShape 的拓扑层次结构详解

## 一、整体层次关系图

```
Compound（复合体）
  └── CompSolid（复合实体）
        └── Solid（实体）
              └── Shell（壳）
                    └── Face（面）
                          └── Wire（环/线框）
                                └── Edge（边）
                                      └── Vertex（顶点）
```

这是一个 **从底向上、逐层组合** 的严格层级结构：

```
Vertex → Edge → Wire → Face → Shell → Solid → CompSolid → Compound
（低层）                                                    （高层）
```

---

## 二、每一层的详细解释

### 1. **Vertex（顶点）** — 最基本的拓扑元素
| 属性 | 说明 |
|------|------|
| **几何含义** | 空间中的一个 **点**（0D） |
| **作用** | 表示边的端点、交点 |
| **示例** | 坐标 `(0, 0, 0)` 处的一个点 |

> Vertex 是拓扑树的 **叶子节点**，没有子元素。

---

### 2. **Edge（边）** — 连接两个顶点的曲线
| 属性 | 说明 |
|------|------|
| **几何含义** | 空间中的一条 **曲线**（1D），如直线、圆弧、B样条曲线 |
| **子元素** | 包含 1~N 个 **Vertex**（起点、终点，可能还有中间参数点） |
| **作用** | 作为 Wire 的组成单元 |
| **示例** | 从 `(0,0,0)` 到 `(1,0,0)` 的一条直线段 |

```
Edge: ─────●────────────●──────
          Vertex(起点)   Vertex(终点)
```

---

### 3. **Wire（线框/环）** — 由边首尾相连组成的闭合或开放路径
| 属性 | 说明 |
|------|------|
| **几何含义** | 一组 **首尾相连的 Edge**（1D） |
| **子元素** | 包含若干 **Edge** |
| **作用** | 作为 Face 的边界 |
| **示例** | 一个矩形的四条边构成一个 Wire |

```
Wire (矩形轮廓):
    ┌────────────┐
    │            │
    │            │
    └────────────┘
   4条 Edge 首尾相连
```

> **关键规则**：Wire 中的 Edge 必须 **共享 Vertex**（相邻边的端点重合），形成一个连续的路径。Wire 可以是 **闭合的**（Outer Wire）也可以是 **开放的**。

---

### 4. **Face（面）** — 由 Wire 界定的曲面区域
| 属性 | 说明 |
|------|------|
| **几何含义** | 空间中的一个 **曲面区域**（2D），如平面、圆柱面、球面 |
| **子元素** | 包含若干 **Wire**（1个外轮廓 + 0~N个内轮廓/孔洞） |
| **作用** | 作为 Shell 的组成单元 |
| **示例** | 一个带圆孔的矩形平面 |

```
Face (带孔的平面):
    ┌────────────┐
    │    ┌──┐    │   外 Wire = 矩形边框
    │    │孔│    │   内 Wire = 圆孔轮廓
    │    └──┘    │
    └────────────┘
```

> **关键规则**：Face 有且仅有 **1 个外 Wire**（Outer Wire），可以有 **0 或多个内 Wire**（Inner Wire，表示孔洞/开槽）。

---

### 5. **Shell（壳）** — 由 Face 拼接而成的连续面片
| 属性 | 说明 |
|------|------|
| **几何含义** | 一组 **通过共享 Edge 连接的 Face**（2D） |
| **子元素** | 包含若干 **Face** |
| **作用** | 作为 Solid 的边界 |
| **示例** | 一个立方体的 6 个面构成一个封闭 Shell |

```
Shell (立方体的表面):
      ┌───┐
     /顶部/|
    ┌───┐ │
    │   │/│
    │   │ /
    └───┘/
    6个 Face 通过共享 Edge 连接
```

> **Shell 可以是封闭的也可以是开放的**：
> - **封闭 Shell**：像一个气球的表面，完全闭合
> - **开放 Shell**：像一张纸，有边界

---

### 6. **Solid（实体）** — 由封闭 Shell 包围的体积区域
| 属性 | 说明 |
|------|------|
| **几何含义** | 空间中的一个 **封闭体积**（3D） |
| **子元素** | 包含若干 **Shell**（1个外 Shell + 0~N个内 Shell/空腔） |
| **作用** | 作为 CompSolid 或 Compound 的组成单元 |
| **示例** | 一个实心球体；一个有内部空腔的箱子 |

```
Solid (空心球):
      ╭──────────╮
      │  ╭────╮  │   外 Shell = 球的外表面
      │  │空腔│  │   内 Shell = 球的内表面
      │  ╰────╯  │
      ╰──────────╯
```

> **关键规则**：Solid 至少包含 1 个 **封闭的外 Shell**。内部的 Shell 代表空腔/空洞。

---

### 7. **CompSolid（复合实体）** — 由共享 Face 的多个 Solid 组成
| 属性 | 说明 |
|------|------|
| **几何含义** | 一组 **通过共享面连接** 的 Solid（3D） |
| **子元素** | 包含若干 **Solid** |
| **作用** | 表示粘合在一起的多个实体（如布尔运算的结果） |
| **示例** | 两个焊接在一起的立方体 |

```
CompSolid:
    ┌──────┬──────┐
    │Solid1│Solid2│
    │      │(共享面)│
    └──────┴──────┘
    两个 Solid 共享中间的 Face
```

> **与 Compound 的区别**：CompSolid 中的 Solid **必须共享拓扑元素**（Face/Edge），而 Compound 中的元素可以完全独立。

---

### 8. **Compound（复合体）** — 最顶层的容器，可以包含任何拓扑对象
| 属性 | 说明 |
|------|------|
| **几何含义** | 任意拓扑元素的 **集合/容器**（0D~3D 混合） |
| **子元素** | 可以包含 Solid、Face、Edge、甚至其他 Compound 等任意类型 |
| **作用** | 最通用的组合方式，用于组合互不连接的对象 |
| **示例** | 螺栓组（每个螺栓是独立 Solid，组合为 Compound） |

```
Compound (散落的零件集合):
    ●  ───  ▲  ■
   点  线段  面  实体
   （各种类型混在一起，互不连接）
```

---

## 三、维度对应总结

| 维度 | 拓扑类型 | 说明 |
|:----:|:--------:|------|
| **0D** | **Vertex** | 点 |
| **1D** | **Edge** → **Wire** | 曲线 → 环路 |
| **2D** | **Face** → **Shell** | 曲面区域 → 面片组 |
| **3D** | **Solid** → **CompSolid** → **Compound** | 体积 → 共享实体组 → 任意组合 |

---

## 四、为什么要这样分层？

### 1. **边界表示法的需要**
整个结构遵循 **B-Rep（Boundary Representation，边界表示法）** 建模范式：

> **一个 n 维的形体，由 (n-1) 维的边界来定义。**

| 形体 | 由什么界定 |
|------|-----------|
| **Solid**（3D体积）| 由封闭的 **Shell**（2D面）界定 |
| **Face**（2D曲面区域）| 由 **Wire**（1D环）界定 |
| **Edge**（1D曲线段）| 由 **Vertex**（0D点）界定 |

### 2. **精确定义拓扑邻接关系**
通过共享低层元素来表达"连接"：
- 两个 Face **共享 Edge** → 它们在 Shell 中相邻
- 两个 Solid **共享 Face** → 它们在 CompSolid 中粘合

### 3. **支持复杂几何操作**
- **布尔运算**（并、交、差）需要精确跟踪拓扑变化
- **倒角/圆角** 操作需要找到共享 Edge 的 Face
- **网格化** 需要遍历 Face → Wire → Edge 来离散化

### 4. **支持非流形几何**
不同层次可以表达：
- 实体、壳体、线框、点云 → 全部统一在 **Compound** 中
- 一个模型可以同时包含实心部分和曲面部分

## 五、一个完整例子：带孔的 L 形实体

```
┌──────┐
│      │
│   ┌──┘
│   │ ○    ← 圆孔穿透实体
│   └──┐
│      │
└──────┘
```

它的拓扑结构：

```
Compound (L形零件)
  └── Solid
        ├── Shell (外表面，由多个 Face 围成)
        │     ├── Face (顶面) → Wire(外轮廓) + Wire(孔轮廓)
        │     ├── Face (底面) → Wire(外轮廓) + Wire(孔轮廓)
        │     ├── Face (侧面1) → Wire(...)
        │     ├── Face (侧面2) → Wire(...)
        │     ├── Face (圆柱孔面) → Wire(...)
        │     └── ... 更多侧面
        └── Shell (内空腔，如果有)
```

每一层拆开：
```
Solid → Shell → Face → Wire → Edge → Vertex
  (1)    (1)     (8)    (16)   (24)   (16)
```

## 六、核心要点速记

> 📌 **记住这个口诀**：
> - **点** 连成 **线**（Vertex → Edge）
> - **线** 围成 **环**（Edge → Wire）
> - **环** 界定 **面**（Wire → Face）
> - **面** 拼成 **壳**（Face → Shell）
> - **壳** 包住 **体**（Shell → Solid）
> - **体** 粘成 **复体**（Solid → CompSolid）
> - 一切皆可装入 **复合体**（→ Compound）

这就是 OpenCascade/FreeCAD 中 **TopoShape 拓扑层次结构** 的完整关系！
