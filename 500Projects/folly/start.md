# Folly 学习笔记

[GitHub - facebook/folly: An open-source C++ library developed and used at Facebook.](https://github.com/facebook/folly)

## 简介

Folly 是 Facebook 开源的 C++ 基础库，提供了大量高性能组件。适合作为进阶 C++ 学习项目。

### 核心模块

- **Future/Promise**：异步编程框架
- **Executor/ThreadPool**：线程池与任务调度
- **ConcurrentHashMap**：高性能并发哈希表
- **fbstring**：优化的字符串实现
- **Arena**：内存分配器
- **Function**：std::function 的优化替代
