1. 安装 MySQL
```shell
sudo apt-get install mysql-server
```
2. 进行初始化配置
```shell
sudo mysql_secure_installation
```
3. 检查 MySQL 状态
```shell
systemctl status mysql.service
```
5. 安装 MySQL 连接库
```shell
apt-get install libmysqlclient-dev
```
