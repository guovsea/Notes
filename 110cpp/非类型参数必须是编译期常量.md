`T:value` 有三种可能

- 类型
- 静态常量 `static const value`
- 静态变量 `static value`

因此 `T:value` 有可能不是编译器常量

```cpp
    template <typename T, bool>  
    struct MyClass {  
    };  
  
    // 错误：偏特化不能依赖于类型的成员  
    template <typename T>  
    struct MyClass<T, T::value> {};  // T::has_value 是在编译时才能知道的，无法用作非类型模板参数
```
