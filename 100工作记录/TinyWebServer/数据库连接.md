## 数据库连接池

保持一定数量的 MySQL 数据库连接，请求到来时，复用已经分配的连接。

数据库连接池应该是**单例模式**，采用的设计方法为：

**local static** 。C++11 规定了 local static 在多线程条件下的初始化行为，要求编译器保证了内部静态变量的线程安全性。因此可以用局部静态变量优雅地实现单例模式。

```cpp
class Singleton
{
private:
	Singleton() { };
	~Singleton() { };
	Singleton(const Singleton&);
	Singleton& operator=(const Singleton&);
public:
	static Singleton& getInstance() 
        {
		static Singleton instance;
		return instance;
	}
};
```
## 数据库连接

利用 RAII 机制，连接对象构造时为其分配一个实际的数据库连接，连接对象释放时回收实际的数据库连接。

```cpp
class connectionRAII{

public:
	connectionRAII(MYSQL **con, connection_pool *connPool);
	~connectionRAII();
	
private:
	MYSQL *conRAII;
	connection_pool *poolRAII;
};

```
