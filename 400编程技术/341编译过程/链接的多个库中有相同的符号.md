此时，会引用优先链接的库中的符号

## 代码

```cpp
# mine.c
int f() {
    return 1;
}
```

```cpp
# your.c
int f() {
    return 0;
}
```

```c
# main.c
#include <printf.h>

int f();
int main(int argc, char const *argv[])
{
    printf("f() = %d\n", f());
    return 0;
}
```

## 实验过程及结果

```shell
➜   clang mine.c -o libmine.so -shared -fPIC
➜   clang your.c -o libyour.so -shared -fPIC
➜   clang main.c -o a.out -lyour -lmine -L.
➜   ./a.out
f() = 0
➜   clang main.c -o a.out -lmine -lyour -L.
➜   ./a.out
f() = 1
```
