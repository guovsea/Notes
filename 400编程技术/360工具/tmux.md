## 概念

**会话** Session

- **会话是 tmux 的最顶层概念，它代表一个完整的工作环境。**
- 会话可以用于持久化地保存工作状态，可以在不同的终端或时间点恢复到同一个会话。

**窗口** Window

- **窗口是会话的一个分割，它是一个工作区域。** 一个工作环境中可以有很多工作区域。
- 窗口通常用于组织不同任务或项目，例如一个窗口用于编写代码，另一个用于运行终端命令。
- 可以在会话中创建多个窗口，然后通过切换窗口来切换不同的任务。

**窗格** Pane

- **窗格是窗口内的分割，是一个工作区域中的不同分区。**
- 一个窗口能够同时显示多个 tty（每个窗格对应一个 tty）
- 窗格可以水平或垂直分割窗口，以便在同一个窗口中同时运行多个命令或查看不同的输出。

## 命令

### Session

在会话中不能创建新的会话、连接到会话

1. **创建新会话** 
    - `tmux new -s session_name`： 创建一个新会话并指定名称。
    - `tmux` ：创建一个新的会话，会以数字命名
2. **杀死会话**：
    - `tmux kill-session -t session_name`: 终止指定名称的会话。
3. **列出会话**：
    - `tmux list-sessions` 或 `tmux ls`: 列出当前所有的会话。
4. **连接到会话**：
    - `tmux attach-session -t session_name` 或
    - `tmux attach -t session_name`: 连接到指定名称的会话。
5. **分离会话**：
    - `Ctrl-b d`：分离当前会话，返回到原始终端。
6. **重命名会话**：
    - `tmux rename-session -t old_name new_name`: 重命名会话。
7. **暂停会话**：
    - `Ctrl-b :` 然后输入 `:suspend-client`：暂停当前会话，使其暂时停止运行，重新进入则会继续进行。
8. **切换会话**：
    - `Ctrl-b (`：切换到上一个会话。
    - `Ctrl-b )`：切换到下一个会话。


### window

1. **创建新窗口**：
    - `Ctrl-b c`：在当前会话内创建一个新窗口。
2. **切换窗口**：
    - `Ctrl-b 0` 到 `Ctrl-b 9`：切换到特定编号的窗口。
    - `Ctrl-b l`  ：切换到上一个窗口。
      *`Ctrl-b Ctrl-h`* ：切换到左边窗户，更符合 vim 逻辑（注意要按住 ctrl 的同时按 h）
    - `Ctrl-b n`：切换到下一个窗口。
      *`Ctrl-b Ctrl-l`*：切换到右边窗口
1. **重命名窗口**：
    - `Ctrl-b ,`：重命名当前窗口。你可以为窗口指定一个描述性的名称，以便更容易识别其内容。
2. **关闭窗口**：
    - `Ctrl-b &`：关闭当前窗口。tmux会提示你确认关闭。
### Pane

1. **垂直分割当前窗格**：
    - `Ctrl-b %`：将当前窗格垂直分割为两个窗格，一个在左边，另一个在右边。
2. **水平分割当前窗格**：
    - `Ctrl-b "`：将当前窗格水平分割为两个窗格，一个在上面，另一个在下面。
3. **切换焦点到下一个窗格**：
    - `Ctrl-b o`：在当前窗格内切换焦点到下一个窗格。
      *`Ctrl-b l ：符合vim逻辑`*
4. **切换到上一个窗格**：
    - `Ctrl-b ;`：在最后两个使用过的窗格之间切换焦点。
      *`Ctrl-b k ：符合vim逻辑`*
2. **交换窗格位置**：
    - `Ctrl-b {`：将当前窗格向前移动，与前一个窗格交换位置。
    - `Ctrl-b }`：将当前窗格向后移动，与后一个窗格交换位置。
3. **关闭当前窗格**：
    - `Ctrl-b x` 或 `Ctrl-d`：关闭当前窗格。tmux会要求你确认关闭。
4. **调整窗格大小**：
    - `Ctrl-b Alt-方向键`：通过按住 `Alt` 键和相应的方向键，可以调整窗格的大小。*没效果，被覆盖了*
      *Ctrl-b K/J/H/L：上下左右拉伸，调整窗格大小*
5. **切换 Pane 布局**：
    - `Ctrl-b 空格`：切换窗格布局，可以在平铺布局、主从布局和平均布局之间切换。
6. **旋转窗格**：
    - `Ctrl-b C-o`：将所有窗格逆时针旋转，光标仍在最开始的位置
7. **将窗格移动到其他窗口**：
    - `Ctrl-b :` 然后输入 `move-pane -t target_window`：将当前窗格移动到指定的目标窗口。
