## 成员函数指针类型

**声明成员函数指针类型**：**`返回值类型 (类类型::*指针名)(形参类型列表)`**

声明普通函数指针类型：`返回值 (*指针名)(参数列表)`

**使用成员函数指针**：**`(对象.*指针名)(实参列表)`**

```cpp
#include <iostream>

class MyClass {
public:
    void memberFunction(int x) {
        std::cout << "Member function called with x = " << x << std::endl;
    }
};

int main() {
    // 创建类的实例
    MyClass myObject;

    // 获取成员函数指针
    void (MyClass::*functionPtr)(int) = &MyClass::memberFunction;

    // 使用成员函数指针调用成员函数
    (myObject.*functionPtr)(42);

    return 0;
}
```
## `类名::名字`并不是成员函数

普通函数函数名可以自动转换为函数指针，`func`等价于`&func`；**但`类名::名字`并不是成员函数指针，只有`&类名::名字`才是成员函数指针**

> 因为`类名::名字`是类的静态数据成员；所以如果使用`类名::名字`会有歧义，因此要使用`&类名::名字`来表示成员函数指针。

```cpp
class MyClass {
public:
    void memberFunction(int x) {
        std::cout << "Member function called with x = " << x << std::endl;
    }
};

int main() {  
    decltype(MyClass::memberFunction) p = MyClass::memberFunction;
    // invalid use of non-static member function ‘void MyClass::memberFunction(int)’
}

// connect(ui->pushButton, QPushButton::clicked, this, MainWindow::testPushBtn);
// error: reference to non-static member function must be called; did you mean to call it with no arguments? 
```
## 区分

- `类名::名字` *静态数据成员*
- `&类名::名字` *成员函数指针，可以是静态的*
- `类名::名字()` *调用静态成员函数*
- `&类名::名字()` *对静态成员函数的返回值取地址*
