谓词(Predicate)指**返回值可转换成bool的可调用对象**

## 一元谓词

接受一个参数的谓词

## 二元谓词

接受两个参数的谓词

**比较谓词**

接受两个参数的的二元谓词，严格**弱序**关系：第一个参数优先于第二个参数则返回true，否则返回false将定义。

```cpp
// 重载sort的比较器，根据a升序，b降序
// NOTE sort的cmp结果为true的话l，r就保持这个顺序（l在左，r在右）
bool cmp(MyStruct l, MyStruct r) {
    if (l.a < r.a) {
        return true;
    } else if (l.a == r.a) {
        return l.b > r.b;
    } else {
        return false;
    }
}
```
