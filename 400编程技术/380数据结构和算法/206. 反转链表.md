题意：反转一个单链表。

示例: 输入: 1->2->3->4->5->NULL 

输出: 5->4->3->2->1->NULL


```cpp
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        // 已经反转好的链表的头部
        ListNode* pre = nullptr;
        // 当前需要修改的节点
        ListNode* cur = head;
        while (cur) {
            // 记录下一个节点
            ListNode* tmp = cur->next;
            cur->next = pre;
            // 顺序不能修改
            pre = cur;
            cur = tmp;
        }
        return pre;
    }
}
```

- 首先定义一个 cur 指针，指向头结点，再定义一个 pre 指针，初始化为 null。

- 然后开始反转了，要保存当前节点的下一个节点，也就是把 cur->next 节点用 tmp 指针保存一下
