## 聚合初始化 Aggregate initialization


是列表初始化的一种方式

只适用于聚合类(aggregate class)

聚合
- array types
- class types that has
  - 没有自定义构造函数
  - 没有虚指针

```cpp
struct S1 { int a, b; };
struct S2 { S1 s, t; };
 
// “x” 的每个子聚合体都有一个附属的以 { 开始的初始化器子句
S2 x[2] =
{
    // 属于 “x[0]”
    {
        {1, 2}, // 属于 “x[0].s”
        {3, 4}  // 属于 “x[0].t”
    },
    // 属于 “x[1]”
    {
        {5, 6}, // 属于 “x[1].s”
        {7, 8}  // 属于 “x[1].t”
    }
};
 
// “x” 和 “y” 的值相同（见下文）
S2 y[2] = {1, 2, 3, 4, 5, 6, 7, 8};
```
